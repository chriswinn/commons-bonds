# Audience Pressure-Test Set Construction (Guide, v3.0)

**Purpose.** Build the acceptance-test character set (Pass 3.3) + the adversarial / detractor character set (Pass 3.4) for a Stage 1 brief, adapted to the specific venue + piece. Referenced from `stage-1-audience-aware-structure-pass.md`.

**Discipline anchor.** v3.0 audience-aware drafting (extends v2.0 Amendment A + Amendment B): the Stage 1 brief includes BOTH the acceptance-test set (Pass 3.3) AND the adversarial-detractor set (Pass 3.4) as the load-bearing audience-load discipline.

Acceptance + robustness are **two distinct tests** with different character sets, different verdict scales, and different diagnostic outputs:

- **Acceptance set** (Pass 3.3): target audiences whose No vote kills the piece (Tier 1) + whose Yes vote amplifies (Tier 2) + whose experience the piece engages (Tier 3). Typically 15-25 characters. Verdict: per-character INCLUDE / NEUTRAL / EXCLUDE.
- **Adversarial set** (Pass 3.4): characters who will TRY to unwind the artifact (industry-funded; predisposed-hostile-by-financial-incentive; ideologically-opposed; trade-press hostile reviewer; etc.). Typically 5-10 characters. Verdict: thread-pull synthesis (the diagnostic is which threads adversarial readers find, NOT pass/fail per character).

## §1. The acceptance-test set (Pass 3.3)

### §1.1 Three-tier structure

| Tier | Description | Typical character count |
|---|---|---|
| **Tier 1 — Gating** | Characters whose No vote kills the piece. Editor + reader at venue; literary agent; acquisitions editor. | 3–5 |
| **Tier 2 — Pipeline-strengthening** | Characters whose Yes vote amplifies the piece. Downstream-essay editors; comp-titles cluster readers; named-tradition successors; field experts. | 4–6 |
| **Tier 3 — Cultural-resonance + accessibility** | Characters whose experience the piece engages or whose accessibility matters. Specific demographic / geographic / professional / lineage groups; general working-class reader; first-gen reader. | 5–9 |

**Total acceptance set:** 12–25 characters. Lighter for short-form pieces (op-eds, defenses); heavier for long-form magazine essays + chapter audits.

### §1.2 Acceptance verdict scale

Per character, score:

- **✓✓✓ INCLUDE** — strong yes; piece lands decisively.
- **✓✓ INCLUDE** — yes; piece lands.
- **✓ INCLUDE** — yes; piece lands with minor reservations.
- **NEUTRAL** — neither alienated nor strongly engaged.
- **⚠ EXCLUDE** — may disengage; flag specific reason.
- **⚠⚠ EXCLUDE** — actively rejects; flag specific reason.
- **⚠⚠⚠ EXCLUDE** — disqualifying; cannot ship to venue with this character as Tier 1.

Aggregate verdict: total INCLUDE vs NEUTRAL vs EXCLUDE + per-tier breakdown. Tier 1 ⚠⚠⚠ blocks ship.

## §2. The acceptance-set tiers

### Tier 1 — the gating characters

For every venue, Tier 1 should include:

1. **The venue's editor** (named if known — Sam Haselby for Aeon; whoever for Noema; etc.). What does this editor read for? Cite published submission guidance + recent published pieces + any direct knowledge.
2. **The venue's typical reader.** Pull from venue's masthead positioning + masthead pieces + reviewer-of-record patterns.
3. **A literary agent.** What signals trade-publisher-viability? What signals "this author can sustain a book"?
4. **An acquisitions editor at a trade publisher.** What signals bookstore-buyer hook, comp-titles, category placement?

For pieces with specific agent / publisher relationships established: name them. Generic "literary agent" is acceptable when no specific person is in flight.

---

### Tier 2 — pipeline-strengthening

Include characters who:

- Edit / review at adjacent venues the book will eventually reach (e.g., for an Aeon piece, include a Boston Review editor; for a Noema piece, include an n+1 editor)
- Hold positions in the lineage the piece engages (e.g., for a piece on intergenerational pricing, include a Stern-tradition economist + a Hartwick-tradition resource economist)
- Are comp-titles cluster readers (e.g., a Mazzucato reader; a Pistor reader)
- Hold prizes / fellowships the piece could attract (e.g., a Berggruen judge; an Aeon Philosophy Prize panel)

Pull names from the cascade plan + comp-titles deep matrix + outreach pipeline.

---

### Tier 3 — cultural-resonance + accessibility

Include characters who:

- Belong to lineages / traditions the piece engages culturally (e.g., for a piece engaging Dunbar / Du Bois / Ellison / Fanon, include a Black reader in that lineage; check named-subject consent discipline implications)
- Live in places the piece references (e.g., for McDowell County content, include a McDowell County resident; for Chesapeake content, include a Fox Hill waterman)
- Hold professional identities the piece references (e.g., for coal-mining content, include a coal miner; for medical-system content, include a health-care worker)
- Represent accessibility cases — general working-class reader; first-gen-reader; immigrant reader; reader-without-cited-prior-knowledge

Tier 3 characters often surface the deepest concerns about a piece's authorial-positioning + lineage-handling. They're where named-subject consent issues + instrumentalization risks surface.

## §3. The adversarial / detractor set (Pass 3.4) — NEW in v3.0

The adversarial set tests **robustness**: do hostile readers find threads that would let them unwind the artifact? Required for any chapter-level artifact (per pipeline doctrine retrofit policy) + any artifact where mainstream-reception scrutiny is plausible.

### §3.1 Adversarial verdict scale (different from acceptance)

Adversarial characters are SELECTED for hostile read. Verdict-floor is EXCLUDE; INCLUDE / NEUTRAL are not in scope. The diagnostic value is in which threads the adversarial characters collectively find.

- **⚠ EXCLUDE** — would push back but the artifact holds against the push-back; thread is acknowledged.
- **⚠⚠ EXCLUDE** — would actively reject; finds load-bearing thread; artifact's structural moves don't disarm.
- **⚠⚠⚠ EXCLUDE** — would weaponize the artifact against itself; finds artifact-disqualifying thread.

### §3.2 Adversarial character types

Build the adversarial set across these categorical hostile-read positions. Typical adversarial set: 5–10 characters drawn from the categories below, adapted to the artifact's specific framework-claims + scope.

#### Industry-funded / financial-incentive adversarial characters

- **Industry-funded energy economist** (Heritage Foundation / Manhattan Institute / Texas Public Policy Foundation / American Petroleum Institute / National Mining Association). Predisposed-hostile by financial-incentive to fossil-fuel-industry economics threat. Reads for threads to weaponize against framework's policy implications.
- **Industry-funded health economist** (pharma-industry-affiliated; tobacco-industry-funded-research-historical-analog). Predisposed-hostile to pharma cost-shifting accounting threats.
- **Industry-funded financial-services economist** (banking / private-equity / asset-management lobby-funded). Predisposed-hostile to financial-regulation framework implications.

#### Ideologically-opposed adversarial characters

- **Public-choice theorist** (Buchanan / Tullock / Virginia school tradition). Adversarial-intellectual. Reads for skipped rent-seeking analysis (Public Choice argument: institutions don't "design themselves" — actors with rent-seeking incentives design them; reads framework's no-villain frames as masking the rent-seekers).
- **Pure-libertarian / Rothbardian reader** (Mises Institute / FEE / Cato-libertarian-wing). Adversarial-ideological. Reads framework as expanded-state-power proposal cloaked in scholarly language; reads bond instruments as state-enforced regulatory architecture.
- **Climate / energy-realist reader** (shale-cluster; AEI energy fellow; Manhattan Institute energy fellow; *RealClearEnergy* reader). Adversarial-energy. Reads commons-pricing framing as policy-attack vector.
- **Orthodox-econ reader** (Chicago School; freshwater macro). Adversarial to heterodox-econ framings + market-failure premises.

#### Trade-press / commercial-adversarial characters

- **Trade-press hostile reviewer** (NYT / WSJ / *The Atlantic* / *The New Republic* book-reviewer assigned with adversarial intent). Adversarial-commercial. Would write a hostile review if assigned. Note: this character's verdict is largely determined by the assignment, not by the chapter; the mitigation is at review-cycle level (supportive reviews from non-hostile venues offset).
- **Wall Street Journal editorial-board reader / business-press conservative.** Adversarial-business-press. Reads framework-promise + no-villain framing as preparation for business-blame regulation.
- **Tabloid / populist-skeptic reader** (NY Post / Daily Mail / Fox News op-ed-page reader). Adversarial-populist. Reads memoir-substrate as celebrity-style legitimation-cover for policy-advocacy; reads specific scene-anchors for populist character-attack ammunition.

#### Cultural-adversarial characters

- **Reactionary intellectual reader** (*National Review* / *National Affairs* / *American Affairs* reader; Niall Ferguson / Bret Stephens / Ross Douthat-cluster). Adversarial-cultural. Reads lineage-engagement (Du Bois / Dunbar / Frankfurt-school adjacent) + architecture-naming as left-coded scholarship; predisposed-suspicious.
- **Religious-traditionalist reader** (First Things / The American Conservative reader). Adversarial-cultural-religious. Reads structural-rather-than-blame framings + market-corrective framework as displacement of moral-agency frame.

#### Legal-adversarial characters

- **Tort-reform / fiduciary-protection lawyer** (defense-bar; corporate-counsel cluster). Adversarial-legal. Reads bond-instruments as litigation-amplification threat; reads framework signals for enforcement-mechanism aggressiveness.

### §3.3 How to select adversarial characters

Adversarial-set selection is artifact-specific. Build the set by asking:

1. **Whose financial incentive does this artifact threaten?** Include industry-funded economists / lobbyists / lawyers / spokespeople aligned with the threatened interest.
2. **Which ideological tradition would read this artifact as positionally hostile?** Include intellectual-tradition adversaries (Public Choice; Rothbardian libertarianism; Chicago-School econ; reactionary intellectual).
3. **Which trade-press / commercial venues would assign a hostile reviewer?** Include trade-press hostile reviewer + business-press conservative + tabloid populist.
4. **Which cultural-tradition adversaries would read this artifact's lineage-engagement as positioning?** Include reactionary intellectual; religious-traditionalist.
5. **Which legal / enforcement adversaries would read this artifact's framework-promise as risk?** Include tort-reform / fiduciary-protection lawyer + corporate-counsel cluster.

Typical full-rigor adversarial set: 5-10 characters drawn from above. Smaller artifacts can scale down (3-5 characters covering the highest-risk hostile-read positions for the specific artifact).

### §3.4 Thread-pull synthesis (canonical Pass 3.4 diagnostic)

After scoring each adversarial character, produce the **thread-pull synthesis table** — the canonical diagnostic output:

| Thread | Pulled by adversarial chars # | Type | Recommendation |
|---|---|---|---|
| [thread description] | [list of char numbers] | Load-bearing chapter claim / Procedural-cosmetic flag | [hold / spot-fix / cross-chapter handoff] |

**Type classification:**

- **Load-bearing chapter claim** — the thread is doing major work for several Tier-1/2 acceptance audiences (e.g., for Ch 1: line 81 no-villain framing is load-bearing for anti-capital, left-policy, structural-critique, civic-republican audiences). Chapter must hold its ground. Spot-fix would damage acceptance verdicts. Recommendation: acknowledge thread + verify structural moves disarm + cross-chapter handoff if thread warrants further engagement at another chapter.
- **Procedural / cosmetic flag** — the thread is a surface-flaw the chapter could disarm without damaging acceptance verdicts. Recommendation: apply spot-fixes.

**Robustness verdict:**

- **ROBUST** — no common load-bearing threads found; isolated procedural flags only.
- **CONDITIONALLY ROBUST** — common load-bearing thread found that the chapter's structural moves do disarm; thread acknowledged in pre-pub review queue.
- **REQUIRES STRUCTURAL ENGAGEMENT** — common load-bearing thread found that the chapter does NOT disarm; cross-chapter workstream or chapter-level structural revision needed.

### §3.5 Worked example — Ch 1 REAUDIT v3 Public Choice thread

From the Ch 1 Pass 3 REAUDIT v3 §5.3 thread-pull synthesis (the canonical example):

> The **line 81 no-villain framing** was pulled by 5 adversarial characters (#31 industry-funded energy economist; #33 Public-Choice theorist; #36 reactionary intellectual; #38 WSJ editorial-board reader; #40 tabloid populist-skeptic) with different pulls per character. The thread was classified as **load-bearing chapter claim** because the structural-rather-than-blame framing is doing major work for several Tier-1/2 acceptance audiences (anti-capital, left-policy, structural-critique, civic-republican). The chapter must hold its ground at Ch 1. **But: Public Choice critique (#33) is the most intellectually-serious thread; the book's analytical chapters need to engage rent-seeking analysis directly somewhere (likely Ch 5 + Ch 9 + Tech Appendix).** Ch 1 spot-fix not warranted (would damage acceptance verdicts); cross-chapter handoff flag — became the cross-chapter rent-seeking-engagement workstream.

This is the canonical adversarial-thread-pull case: a thread that surfaces as load-bearing at one chapter but warrants cross-chapter engagement; resolved via a cross-chapter workstream per pipeline doctrine §5.

## §4. Venue-specific adaptations (acceptance set)

### Aeon (literary-philosophical)
- **Tier 1:** Sam Haselby (philosophy editor); Aeon reader (philosophy-curious general intellectual); literary agent (Wylie cluster); acquisitions editor (trade-press philosophical).
- **Tier 2:** Aeon Philosophy Prize judges (Warburton / Dresser); Mazzucato/Raworth-cluster reader; Coasean economist (devil's-advocate); analytic-philosophy reader.
- **Tier 3:** general intellectual without prior-Commons-Bonds-knowledge; first-gen reader; non-Western reader (analytic philosophy can read as Western-academic by default).

### Noema (memoir + history/theory + reportage)
- **Tier 1:** Noema editorial brain (three-register weave + low jargon tolerance + named-sources-with-cited-public-record); Noema reader (politically aware intellectual); literary agent; acquisitions editor.
- **Tier 2:** Berggruen Prize judge; Mazzucato/Raworth-cluster reader; Ostrom-successor / institutional analyst; Coasean economist.
- **Tier 3:** Black readers in Dunbar/Du Bois/Fanon/Ellison lineage; Indigenous readers; specific-region readers (Chesapeake / McDowell); McDowell County resident; working-class reader; first-gen immigrant.

### Boston Review (institutional / policy-centrist)
- **Tier 1:** BR editor (policy-economics-curious); BR reader (centrist-policy-intellectual); literary agent; acquisitions editor.
- **Tier 2:** policy-economics reader (Brookings / PIIE register); accountability-tradition successor (Pistor / Christophers); Berggruen judge.
- **Tier 3:** general policy-curious reader; specific demographic groups affected by the piece's case studies.

### Phenomenal World (heterodox-econ academic-adjacent)
- **Tier 1:** PW editor; PW reader (heterodox-econ-curious academic-adjacent intellectual); literary agent; acquisitions editor; **CENTER-RIGHT POLICY READER** (Reason / National Review / AEI-econ register — for non-partisan framing pressure-test per the $100 Barrel Q1 rigor pass Condition 1).
- **Tier 2:** Stone Center reviewer; JFI-network reader; IIPP-network reader; Stern-tradition economist; Hartwick-tradition resource economist.
- **Tier 3:** general policy-economist; non-Anglo reader; working-class reader.

### Project Syndicate / Bloomberg Opinion (short-form policy / financial)
- **Tier 1:** PS/Bloomberg editor; news-pegged reader; literary agent (less load-bearing for op-eds); acquisitions editor (less load-bearing).
- **Tier 2:** international policy reader; financial-press reader; comp-titles cluster reader.
- **Tier 3:** general financial-curious reader; specific affected demographic groups.

### NYT / WaPo Opinion (stretch — general educated reader)
- **Tier 1:** opinion-section editor; NYT general reader; literary agent; acquisitions editor.
- **Tier 2:** intellectually-engaged policy reader; named-tradition successor.
- **Tier 3:** general lay-reader (NYT educated-public); reader unfamiliar with framework apparatus.

### Real-World Economics Review (heterodox-econ methods-note)
- **Tier 1:** RWER editor; heterodox-econ academic; literary agent (less); acquisitions editor (less — academic-press track).
- **Tier 2:** Stone Center / JFI / IIPP-network reader; orthodox-econ reader (devil's-advocate).
- **Tier 3:** policy-econ reader; non-academic intellectual.

### Trade-publisher acquiring editor (book proposal review)
- **Tier 1:** acquiring editor at target imprint; trade-publishing-conventional reader; commercial-arc evaluator; comp-titles evaluator.
- **Tier 2:** comp-titles cluster readers; named-blurber candidates; review-outlet evaluators.
- **Tier 3:** bookstore buyer (commercial signal); first-gen book buyer.

## §5. Stage 0 carry-forward conditions → pressure-test additions

If a Stage 0 publishing-strategy rigor pass returned CONDITIONAL with carry-forward conditions, embed each condition as an additional pressure-test character in the Stage 1 brief's §1.

Example (from $100 Barrel Q1 rigor pass Condition 1 — non-partisan framing discipline):

> **Tier 1 (additional) — Center-right policy reader.** *Reason* magazine / *National Review* / AEI-econ register. Reads §III for partisan tells. Flags: Marx framing (lineage-map progression); Rawls applied-against-discounting (philosophical commitment); implicit-Stern-endorsement (academic-left positioning). Test: does the piece read as "scholarly inventory" (Yes vote) or "authorial commitment" (No vote)? Stage 3 audience-load pass verdict here is the dispositive non-partisan-framing test.

This character then runs through Stage 3's audience-load pass as a Tier 1 gating audience.

## §6. Character sketch template

For each character, provide a ~50-word sketch:

```
**[Character name or descriptor]** ([tier])
- Background: [profession / institution / demographic / role]
- Expected register: [what they read for, what makes them say Yes]
- Likely concerns: [what would make them say No or trip them up]
- Test for this audience: [specific spot-check to apply in Stage 3]
```

Example (Noema editor):

> **Noema editorial brain** (Tier 1 — gating)
> - Background: Editor at Noema Magazine. Reads pitches in days; full essays in minutes-to-an-hour.
> - Expected register: Three-register weave (memoir + history/theory + reportage). Named living sources with cited public-record authority. Concrete numbers + named institutions. Internationalist scope. Low jargon tolerance. Length within venue flex.
> - Likely concerns: Apparatus jargon ("cluster-γ" / "RCV"); generic-intellectual register without specific anchoring; missing named-source authority for cases.
> - Test: does the piece weave three registers (memoir + history/theory + reportage)? Does every case have at least one named living source with cited public-record attribution? Is the piece's framework legible in plain language (no Greek letters, no acronyms)?

## §7. Common pitfalls

1. **Treating the venue's reader as monolithic.** Venues have distinct editor-brain, masthead-reader, and reviewer pools. Build characters for each.
2. **Stuffing Tier 3 with too many overlapping characters.** Each character should test a distinct concern. If two characters return identical findings, collapse them.
3. **Skipping the named-subject consent discipline.** If a Tier 3 character is from a community the piece names by name, the audience-load test interlocks with the consent discipline.
4. **Generic "literary agent" without specificity.** When a specific agent (e.g., Wylie / Sarah Chalfant or Tracy Bohan) is in pipeline, name them. The character sketch should reflect their published representation register.
5. **Building a set that's all-positive.** A useful set includes characters likely to say No or trip up. Set that returns 14/14 INCLUDE on first draft probably wasn't built rigorously.

## §8. Quick-start: defaults by piece type

### Acceptance set (Pass 3.3) defaults

| Piece type | Tier 1 | Tier 2 | Tier 3 | Total acceptance |
|---|---|---|---|---|
| Chapter (book-internal artifact, full rigor) | 5 | 8 | 12-17 | 25-30 |
| Magazine essay (3,000–5,000w) | 4 | 5 | 7 | 16 |
| Magazine pitch (250–500w) | 3 | 3 | 4 | 10 |
| Op-ed (700–1,200w) | 3 | 3 | 4 | 10 |
| In-book defense paragraph (200–400w) | 3 | 3 | 4 | 10 |
| Book-proposal section (1–2pp) | 3 | 3 | 3 | 9 |
| Pitch letter / cover letter | 2 | 1 | 1 | 4 |
| Academic-policy methods note | 4 | 4 | 3 | 11 |

| Piece type | Industry-funded | Ideologically-opposed | Trade-press | Cultural | Legal | Total adversarial |
|---|---|---|---|---|---|---|
| Chapter (book-internal artifact, full rigor) | 2 | 3 | 3 | 1-2 | 1 | 10 |

Adjust based on venue + specific piece + Stage 0 carry-forward conditions + artifact's specific framework-claims that map to specific hostile-read positions.
