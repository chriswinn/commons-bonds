## The three-tier structure

| Tier | Description | Typical character count |
|---|---|---|
| **Tier 1 — Gating** | Characters whose No vote kills the piece. Editor + reader at venue; literary agent; acquisitions editor. | 3–5 |
| **Tier 2 — Pipeline-strengthening** | Characters whose Yes vote amplifies the piece. Downstream-essay editors; comp-titles cluster readers; named-tradition successors; field experts. | 4–6 |
| **Tier 3 — Cultural-resonance + accessibility** | Characters whose experience the piece engages or whose accessibility matters. Specific demographic / geographic / professional / lineage groups; general working-class reader; first-gen reader. | 5–9 |

**Total:** 12–20 characters. Lighter for short-form pieces (op-eds, defenses); heavier for long-form magazine essays.

---

## Tier 1 — the gating characters

For every venue, Tier 1 should include:

1. **The venue's editor** (named if known — Sam Haselby for Aeon; whoever for Noema; etc.). What does this editor read for? Cite published submission guidance + recent published pieces + any direct knowledge.
2. **The venue's typical reader.** Pull from venue's masthead positioning + masthead pieces + reviewer-of-record patterns.
3. **A literary agent.** What signals trade-publisher-viability? What signals "this author can sustain a book"?
4. **An acquisitions editor at a trade publisher.** What signals bookstore-buyer hook, comp-titles, category placement?

For pieces with specific agent / publisher relationships established: name them. Generic "literary agent" is acceptable when no specific person is in flight.

---

## Tier 2 — pipeline-strengthening

Include characters who:

- Edit / review at adjacent venues the book will eventually reach (e.g., for an Aeon piece, include a Boston Review editor; for a Noema piece, include an n+1 editor)
- Hold positions in the lineage the piece engages (e.g., for a piece on intergenerational pricing, include a Stern-tradition economist + a Hartwick-tradition resource economist)
- Are comp-titles cluster readers (e.g., a Mazzucato reader; a Pistor reader)
- Hold prizes / fellowships the piece could attract (e.g., a Berggruen judge; an Aeon Philosophy Prize panel)

Pull names from the cascade plan + comp-titles deep matrix + outreach pipeline.

---

## Tier 3 — cultural-resonance + accessibility

Include characters who:

- Belong to lineages / traditions the piece engages culturally (e.g., for a piece engaging Dunbar / Du Bois / Ellison / Fanon, include a Black reader in that lineage; check named-subject consent discipline implications)
- Live in places the piece references (e.g., for McDowell County content, include a McDowell County resident; for Chesapeake content, include a Fox Hill waterman)
- Hold professional identities the piece references (e.g., for coal-mining content, include a coal miner; for medical-system content, include a health-care worker)
- Represent accessibility cases — general working-class reader; first-gen-reader; immigrant reader; reader-without-cited-prior-knowledge

Tier 3 characters often surface the deepest concerns about a piece's authorial-positioning + lineage-handling. They're where named-subject consent issues + instrumentalization risks surface.

## Venue-specific adaptations

### Aeon (literary-philosophical)
- **Tier 1:** Sam Haselby (philosophy editor); Aeon reader (philosophy-curious general intellectual); literary agent (Wylie cluster); acquisitions editor (trade-press philosophical).
- **Tier 2:** Aeon Philosophy Prize judges (Warburton / Dresser); Mazzucato/Raworth-cluster reader; Coasean economist (devil's-advocate); analytic-philosophy reader.
- **Tier 3:** general intellectual without prior-Commons-Bonds-knowledge; first-gen reader; non-Western reader (analytic philosophy can read as Western-academic by default).

### Noema (memoir + history/theory + reportage)
- **Tier 1:** Noema editorial brain (three-register weave + low jargon tolerance + named-sources-with-cited-public-record); Noema reader (politically aware intellectual); literary agent; acquisitions editor.
- **Tier 2:** Berggruen Prize judge; Mazzucato/Raworth-cluster reader; Ostrom-successor / institutional analyst; Coasean economist.
- **Tier 3:** Black readers in Dunbar/Du Bois/Fanon/Ellison lineage; Indigenous readers; specific-region readers (Chesapeake / McDowell); McDowell County resident; working-class reader; first-gen immigrant.

### Boston Review (institutional / policy-centrist)
- **Tier 1:** BR editor (policy-economics-curious); BR reader (centrist-policy-intellectual); literary agent; acquisitions editor.
- **Tier 2:** policy-economics reader (Brookings / PIIE register); accountability-tradition successor (Pistor / Christophers); Berggruen judge.
- **Tier 3:** general policy-curious reader; specific demographic groups affected by the piece's case studies.

### Phenomenal World (heterodox-econ academic-adjacent)
- **Tier 1:** PW editor; PW reader (heterodox-econ-curious academic-adjacent intellectual); literary agent; acquisitions editor; **CENTER-RIGHT POLICY READER** (Reason / National Review / AEI-econ register — for non-partisan framing pressure-test per the $100 Barrel Q1 rigor pass Condition 1).
- **Tier 2:** Stone Center reviewer; JFI-network reader; IIPP-network reader; Stern-tradition economist; Hartwick-tradition resource economist.
- **Tier 3:** general policy-economist; non-Anglo reader; working-class reader.

### Project Syndicate / Bloomberg Opinion (short-form policy / financial)
- **Tier 1:** PS/Bloomberg editor; news-pegged reader; literary agent (less load-bearing for op-eds); acquisitions editor (less load-bearing).
- **Tier 2:** international policy reader; financial-press reader; comp-titles cluster reader.
- **Tier 3:** general financial-curious reader; specific affected demographic groups.

### NYT / WaPo Opinion (stretch — general educated reader)
- **Tier 1:** opinion-section editor; NYT general reader; literary agent; acquisitions editor.
- **Tier 2:** intellectually-engaged policy reader; named-tradition successor.
- **Tier 3:** general lay-reader (NYT educated-public); reader unfamiliar with framework apparatus.

### Real-World Economics Review (heterodox-econ methods-note)
- **Tier 1:** RWER editor; heterodox-econ academic; literary agent (less); acquisitions editor (less — academic-press track).
- **Tier 2:** Stone Center / JFI / IIPP-network reader; orthodox-econ reader (devil's-advocate).
- **Tier 3:** policy-econ reader; non-academic intellectual.

### Trade-publisher acquiring editor (book proposal review)
- **Tier 1:** acquiring editor at target imprint; trade-publishing-conventional reader; commercial-arc evaluator; comp-titles evaluator.
- **Tier 2:** comp-titles cluster readers; named-blurber candidates; review-outlet evaluators.
- **Tier 3:** bookstore buyer (commercial signal); first-gen book buyer.

## Stage 0 carry-forward conditions → pressure-test additions

If a Stage 0 publishing-strategy rigor pass returned CONDITIONAL with carry-forward conditions, embed each condition as an additional pressure-test character in the Stage 1 brief's §1.

Example (from $100 Barrel Q1 rigor pass Condition 1 — non-partisan framing discipline):

> **Tier 1 (additional) — Center-right policy reader.** *Reason* magazine / *National Review* / AEI-econ register. Reads §III for partisan tells. Flags: Marx framing (lineage-map progression); Rawls applied-against-discounting (philosophical commitment); implicit-Stern-endorsement (academic-left positioning). Test: does the piece read as "scholarly inventory" (Yes vote) or "authorial commitment" (No vote)? Stage 3 audience-load pass verdict here is the dispositive non-partisan-framing test.

This character then runs through Stage 3's audience-load pass as a Tier 1 gating audience.

## Character sketch template

For each character, provide a ~50-word sketch:

```
**[Character name or descriptor]** ([tier])
- Background: [profession / institution / demographic / role]
- Expected register: [what they read for, what makes them say Yes]
- Likely concerns: [what would make them say No or trip them up]
- Test for this audience: [specific spot-check to apply in Stage 3]
```

Example (Noema editor):

> **Noema editorial brain** (Tier 1 — gating)
> - Background: Editor at Noema Magazine. Reads pitches in days; full essays in minutes-to-an-hour.
> - Expected register: Three-register weave (memoir + history/theory + reportage). Named living sources with cited public-record authority. Concrete numbers + named institutions. Internationalist scope. Low jargon tolerance. Length within venue flex.
> - Likely concerns: Apparatus jargon ("cluster-γ" / "RCV"); generic-intellectual register without specific anchoring; missing named-source authority for cases.
> - Test: does the piece weave three registers (memoir + history/theory + reportage)? Does every case have at least one named living source with cited public-record attribution? Is the piece's framework legible in plain language (no Greek letters, no acronyms)?

## Common pitfalls

1. **Treating the venue's reader as monolithic.** Venues have distinct editor-brain, masthead-reader, and reviewer pools. Build characters for each.
2. **Stuffing Tier 3 with too many overlapping characters.** Each character should test a distinct concern. If two characters return identical findings, collapse them.
3. **Skipping the named-subject consent discipline.** If a Tier 3 character is from a community the piece names by name, the audience-load test interlocks with the consent discipline.
4. **Generic "literary agent" without specificity.** When a specific agent (e.g., Wylie / Sarah Chalfant or Tracy Bohan) is in pipeline, name them. The character sketch should reflect their published representation register.
5. **Building a set that's all-positive.** A useful set includes characters likely to say No or trip up. Set that returns 14/14 INCLUDE on first draft probably wasn't built rigorously.

## Quick-start: defaults by piece type

| Piece type | Tier 1 | Tier 2 | Tier 3 | Total |
|---|---|---|---|---|
| Magazine essay (3,000–5,000w) | 4 | 5 | 7 | 16 |
| Magazine pitch (250–500w) | 3 | 3 | 4 | 10 |
| Op-ed (700–1,200w) | 3 | 3 | 4 | 10 |
| In-book defense paragraph (200–400w) | 3 | 3 | 4 | 10 |
| Book-proposal section (1–2pp) | 3 | 3 | 3 | 9 |
| Pitch letter / cover letter | 2 | 1 | 1 | 4 |
| Academic-policy methods note | 4 | 4 | 3 | 11 |

Adjust based on venue + specific piece + Stage 0 carry-forward conditions.
