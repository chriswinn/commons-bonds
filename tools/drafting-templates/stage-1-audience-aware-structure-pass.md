## Customization notes

- For **short-form pieces (<1,500w)**: scale §1 audience set down to 8–12 characters; scale §4 structural decisions down to paragraph-level instead of section-level; keep §6 canonical-facts inventory at full depth (factual drift is the same risk regardless of length).
- For **derivative pieces from withdrawn essays**: the §III material from `archive/_OneDayMaybe/` is the canonical-facts source; the prose paragraphs are explicitly NOT carried forward (they're internal-scaffolding from prior voice).
- For **in-book defense paragraphs (~200–300w)**: this template is overkill. Use an inline mini-brief: 2–3 sentences of context + 1-paragraph canonical-facts note + the bond-defense / flagship-terms model artifacts as a reference. Then bundle Stage 2 + Stage 3 in a single session.
- For **pieces with named subjects**: the named-subject consent discipline applies. Verify consent status in §6 canonical-facts inventory; flag any unsigned-consent subjects for anonymization in Stage 2.
