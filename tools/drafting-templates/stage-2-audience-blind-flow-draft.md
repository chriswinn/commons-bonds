# Stage 2 — Audience-Blind Flow Draft (Template)

**Purpose.** Fresh-session drafting from the Stage 1 brief alone. Audience-blind = the drafter does NOT read source-prose paragraphs, only the Stage 1 brief's canonical-facts inventory + structural decisions + voice register.

**When to use.** Required for any publisher-facing prose >~500w that derives from chapter / source prose (Path B preemptive policy applies). Bundled with Stage 1 only for in-book defense paragraphs.

**Critical experimental constraint.** Stage 2 MUST be a fresh session — not a continuation of Stage 1. The audience-blind discipline is the load-bearing experimental claim of v2.0; bundling with Stage 1 contaminates the test.

**Canonical examples.**
- Long-form essay Stage 2: [`publishing/essays/noema-commons-bonds/essay.md`](../../publishing/essays/noema-commons-bonds/essay.md)
- Short-form pitch Stage 2: [`publishing/essays/aeon-mask-of-abundance/essay.md`](../../publishing/essays/aeon-mask-of-abundance/essay.md)

---

## How to use this template

1. Confirm Stage 1 brief is ratified by author before firing Stage 2.
2. Fill in the `[BRACKET PLACEHOLDERS]` below.
3. Paste into a FRESH session (new chat, new worktree). Stage 2 cannot continue Stage 1.
4. Session produces the draft artifact.
5. STOP at draft; Stage 3 is a separate session.

## Customization notes

- **For in-book defense paragraphs (~200–300w)**: this template is overkill. Use the bundled-Stage 1+2 model from the bond-defense / flagship-terms workstreams. Reference `tools/drafts/why-bonds-paragraph_2026-05-11_v1.0.0.md` for the bundled-session pattern.
- **For derivative pieces from withdrawn essays**: list the withdrawn essay file as a "DO NOT OPEN as source text" file. The Stage 1 brief §6 carries the canonical facts; Stage 2 drafts fresh prose.
- **For pieces where the prior approach had a Path B contamination**: this is exactly the regime the v2.0 discipline was designed for. The "do-not-open" list must include any prior-approach drafts of the same piece (e.g., Essay A when drafting Essay B as Path B's clean alternative).

---

## What Stage 2 does NOT do

- Does NOT evaluate its own output.
- Does NOT run Stage 3 rigor passes.
- Does NOT iterate / revise / spot-fix.
- Does NOT propose meta-conclusions about the discipline framework.
- Does NOT contact named subjects.
- Does NOT modify Stage 1 brief content.

If Stage 2 surfaces a gap in the Stage 1 brief (e.g., a canonical fact missing for a scene the structural plan requires), it should flag the gap in its session-end report and STOP, not patch around it. The author then decides whether to amend Stage 1 brief + re-fire Stage 2, or proceed with the gap acknowledged.
