Ring 1 — Public Adoption Targets (load-bearing for framework durability)

These terms travel outside the book into legal / policy / academic discourse. The framework's adoption-durability bet rests on these. Seven Ring-1 terms ratified as of 2026-04-24.

Ring 1 · Framework name + book title
Commons Bonds
The framework's name + the book's title. Plural "Commons" is load-bearing: the framework operates across MULTIPLE commons categories (the 10 examples surfaced via CIT). The bonds are the obligations owed to shared resources + the financial instruments (Restitution Bond + Foreclosure Bond = aggregate Accountability Bond) that could enforce these obligations. Polysemic title: obligations (commons bonds) becoming shared identity (common bonds).
Ring 1 · Causal-chain event
Value Extraction
The act by which value is separated from its source (a community, ecosystem, or future generation) and taken by a value-capturer. The causal event that produces Cost Severance — extraction severs cost from the capturer (who benefits) and leaves it with the source (who bears). First position in the framework's causal chain: Value Extraction → Cost Severance → Severed Cost. Framework EXTENDS Mariana Mazzucato's value-extraction critique (Mazzucato 2018) — developed there for rent-seeking in financial capitalism — into the physical-resource domain. Echoes David Harvey's accumulation by dispossession (Harvey 2003) rendered in accessible accounting-register language.
Lineage: Mazzucato 2018, The Value of Everything; Harvey 2003, The New Imperialism.
Ring 1 · Mechanism noun
Cost Severance
The structural mechanism by which extraction separates value capture from cost bearing. The process/mechanism noun naming the framework's core phenomenon. Middle position in causal chain: Value Extraction → Cost Severance → Severed Cost. Quantified via the flagship equation CS = RCV − B. Specialization (not contradiction) of the familiar employment-severance concept — inverted to describe the relationship between value-capturers and the commons they draw from. Ch 1 opening builds the rhetorical bridge from HR-severance familiarity to framework usage (per Option C ratification 2026-04-24). Cost severance requires distance between value-capturer and cost-bearer — when that distance collapses to zero, cost severance approaches zero automatically.
Ring 1 · Result noun · flagship adoption target
Severed Cost
A specific cost that has been severed — value extracted by capturer while the cost remains at the source (community, ecosystem, future generation). The result/quantity noun naming the quantified outcome of the Cost Severance mechanism. Flagship adoption target for the framework's success criterion (e.g., labor lawyer uses "severed cost" in a brief).
Ring 1 · Quantification
Residual Commons Value (RCV)
The true intergenerational cost of extraction: foreclosure cost plus externality tail, weighted by substitutability and discounted across time. The price that includes what markets structurally exclude. Forward-looking measurement instrument (companion to backward-looking CSD per Two-Instrument Architecture). Formally: RCV(R, t₀) = ∫[t₀ to ∞] { [1 − S(t|t₀)] · U(R, t, Q(t)) + E(R, t) } · D(t, t₀) dt. The name itself encodes the framework's commons-structural-identity: Residual Commons Value.
Ring 1 · Methodology
Commons Inversion Test (CIT)
Framework-specific Ring-1 methodology for admitting candidate costs to RCV. Renamed 2026-04-24 from Abundance Inversion Test (AIT) per author insight that the test operates on commons (concrete shared resources), not abstract abundances. CIT operates on commons categories: imagine the commons inverted (either absent or unconsumed); if the candidate cost becomes visible as commons-dependent, admit to Cᵢ via Four Gates.

Two sub-forms (formalized as methodology contribution beyond rename):

  • Commons-Absence Inversion — imagine the commons not existing (paradigm: oxygen in space; McDowell habitability destroyed by coal extraction). Operational question: what would it take to replace the commons?
  • Commons-Consumption Inversion — imagine the commons not being consumed (paradigm: commute time returned; forest uncut; medical-commons of dying-with-dignity). Operational question: what could the commons do if not drawn down?

Specialized extension of commons-governance methodology (Ostrom-lineage tradition) to extraction-contexts where commons are drawn down rather than managed-in-place.

Lineage: Ostrom 1990, Hess & Ostrom 2007 (knowledge commons); counterfactual reasoning + minimax-regret (Savage 1951; Loomes-Sugden 1982).
Ring 1 · Cost variables
Cost (Cᵢ)
Indexed cost variables admitted to RCV via the Four Gates (Path F + tier-dissolution ratifications retire the prior 8-tier decomposition; costs are now Cᵢ admitted variables, not pre-sorted tier categories). Each Cᵢ is a specific cost variable with units, time-domain, and admission justification. Each must pass all four Gates: CIT (commons-grounded?) + Units Consistency + Boundedness + Independence.

Ring 2 — Internal Load-Bearing (framework-coherence; not adoption targets)

Readers encounter these to understand the framework's internal structure; non-readers don't adopt them as travel-ready vocabulary. Ten Ring-2 terms ratified as of 2026-04-24.

Ring 2 · Equation
CS = RCV − B (Cost Severance equation)
The framework's central relational claim — Cost Severance equals specifically Residual Commons Value minus Accountability Bond. Three independent contributions: (i) only RCV and B matter for computing CS; (ii) accountability-gap is linear in B; (iii) B = RCV defines the framework's ideal state quantitatively (CS = 0 ↔ honest accounting). With B = B1 + B2 decomposition: total CS = (CSD − B1) + (RCV − B2) = (CSD + RCV) − (B1 + B2). Standard gap-accounting form (Pigou 1920 + Stern 2007 + Nordhaus DICE) with framework-specific decomposition.
Ring 2 · Backward-looking instrument
Cost Severance Damages (CSD)
Backward-looking framework instrument measuring realized human harm from cost severance — health damages, intergenerational disadvantage, community-level suffering, foreclosed life-trajectories. Distinct from RCV (forward-looking resource permanent-foreclosure measure). Pairs with Restitution Bond (B1) in two-instrument architecture. Adopts reparations-economics methodology (Darity & Mullen 2020) + extends to extraction-community contexts.
Lineage: Darity & Mullen 2020; VSL/DALY methodology; Chetty et al. intergenerational mobility; Bullard 1990 environmental justice.
Ring 2 · Aggregate accountability instrument
Accountability Bond (B)
Aggregate framework instrument B = B1 + B2 closing the Cost Severance gap. Every mechanism that forces extractors to bear costs. Decomposed (Block 1 Two-Instrument Architecture):
  • B1 = Restitution Bond (backward-looking; closes CSD gap; reparations + welfare-state + environmental-justice damages compensation + restitution-law tradition).
  • B2 = Foreclosure Bond (forward-looking; closes RCV gap; Hartwick savings + sovereign wealth funds + reclamation bonds + EIBs + Pigouvian carbon taxes). Term-pair coherence with Foreclosure Cost (C₁ in RCV integrand).

The aggregate quantity that should equal RCV for honest accounting. Norway's Government Pension Fund Global is canonical existing B2 exemplar (operationalizes Hartwick rule); B1 globally remains underdeveloped (reparations-economics is the active scholarly territory).

Lineage (B1): Darity & Mullen 2020; Bullard 1990; restitution-law tradition. Lineage (B2): Hotelling 1931, Hartwick rule 1977, Solow 1974, Yang & Davis 2021, Balboa 2016, GAO-17-207R, OSMRE, Pigou 1920.
Ring 2 · Hotelling-extension identity
Hotelling Identity
The framework's articulation that the gap between commons' true cost (RCV) and extractor's market-priced scarcity premium (Hotelling rent) IS, by definition, the per-unit Cost Severance: RCV − Hotelling rent = CS per unit. Bridge between framework vocabulary and standard resource-economics math. Hotelling's part (Hotelling 1931): resource rent = market price − extraction cost = extractor's scarcity premium. Framework's part: defining RCV as commons' true cost; recognizing Hotelling rent under honest accounting represents commons' scarcity value being APPROPRIATED by extractor (rather than compensated to commons); articulating the identity itself. Honest extension; Hotelling didn't write the identity (didn't have RCV).
Lineage: Hotelling 1931 (foundational); Hartwick 1977; Solow 1974; Pearce-Atkinson 1993; Daly stock-flow accounting.
Ring 2 · Estimation methodology
Three Ways of Counting
Three-method estimation methodology for RCV producing a defensible range rather than a single point estimate. The methodology's output is described inline as "the triangulated three-method range."
  • Method 1 — Replacement Cost (lower bound): what would it cost to provide the same service from the next-best substitute? Captures only substitutable component.
  • Method 2 — Norway Revealed Preference (middle-anchor within triangulation): Norway sovereign-fund per-barrel value as anchor. Sidesteps the discount-rate debate. Per Open Insight #14 epistemic humility, Norway didn't fully capture RCV either; functions as middle-anchor between Methods 1 and 3, not absolute middle of true RCV.
  • Method 3 — Scarcity-Adjusted Option Value (upper bound): real-options theory (Dixit-Pindyck 1994) specialized for declining substitutability + future-use uncertainty. Parameters σ + α + V_option.

Convergence reduces uncertainty; divergence reveals where real disagreements live (substitutability vs scarcity-rate vs option-value parameter dominance per sensitivity analysis).

Lineage: Costanza 1997 (Method 1); Hartwick 1977 + Solow 1974 (Method 2); Dixit-Pindyck 1994 (Method 3).
Ring 2 · Substitutability primitive
Substitutability Function S(t|t₀)
Probability function — probability that a functionally adequate substitute for resource R exists at future time t, conditional on baseline technological state at t₀. Appears in foreclosure-cost term C₁ = [1 − S(t|t₀)] · U(R, t, Q(t)). Framework's most novel single mathematical component. Influenced by extraction-timing feedback: earlier extraction can reduce innovation pressure and lower S(t) for future t. Distinct from Hicks 1932 elasticity of substitution (point measure) + Dixit-Pindyck 1994 real options (no commons-extraction context).
Lineage: Hicks 1932 (elasticity of substitution); Dixit-Pindyck 1994 (real-options theory); intertemporal substitution in macro theory.
Ring 2 · Externality primitive
Externality Tail E(R, t)
Time-indexed cost function — damage from extraction of resource R at time t that persists independently of substitute availability. Sibling to foreclosure cost C₁ in the RCV integrand; runs on its own clock (temporally independent of S); structurally specializes Pigouvian externality concept with post-extraction persistence + substitutability-independence. "A substitute for coal doesn't clean up the mine. A substitute for oil doesn't remove carbon from the atmosphere."
Lineage: Pigou 1920 (foundational); Nordhaus DICE; Stern Review 2007.
Ring 2 · Diagnostic ratio
Intergenerational Pricing Gap (IPG)
Ratio IPG = RCV / market_price — the factor by which markets underprice extraction at the point of transaction. "IPG = 33" means the market price covers approximately 3% of the true intergenerational cost. Framework-specific market-pricing-gap diagnostic, distinct from Cost Severance (which uses B not market_price). McDowell coal IPG = 33 is framework's canonical empirical compression. By construction IPG > 1 for extraction cases where RCV captures intergenerational costs markets don't price; empirical work quantifies MAGNITUDE.
Ring 2 · Mechanism (paired with CIT)
Abundance Masking
The mechanism by which abundance makes costs invisible to market pricing until scarcity forces them into view. Oxygen feels free because Earth's atmosphere is abundant. Ecological services feel free because ecosystems haven't collapsed yet. Paired with CIT: masking is the process; CIT is the methodology that detects what masking hides. Promoted to Ring 2 from earlier demote-to-prose ratification per re-examination 2026-04-24 — framework benefits from naming both sides of the mechanism + methodology pair (analogous to Cost Severance + Severed Cost).
Ring 2 · Decision rule
Asymmetric Regret Rule (ARR)
Framework-specific operational decision rule — when RCV tails cannot be precisely quantified, default to the option whose action is reversible. Aphorism: "We can always extract later. We can never un-extract." Specialization of Savage 1951 minimax-regret with irreversibility-weighted regret tails. Bidirectional flip-by-context (per Working Principle #5): directs extract-aggressively (Comet flyby — access window closes; extraction is the reversible choice) AND preserve (Europa — incommensurable biosphere externality; preservation is the reversible choice). Renamed 2026-04-24 from Asymmetric Regret Principle — last-word "Principle" → "Rule" to downgrade overclaim while preserving regret-theory direct lineage in name.
Lineage: Savage 1951 (minimax regret); Loomes-Sugden 1982 (regret theory); Lempert-Popper-Bankes 2003 RDM; Rio Declaration 1992 (precautionary principle adjacent).

Architecture — Two-Instrument + Four Gates

Two-Instrument Architecture

The framework measures two orthogonal aspects of an extraction event:

  • CSD (Cost Severance Damages) — backward-looking; realized human harm; reparations-economics-grounded; pairs with B1 (Restitution Bond).
  • RCV (Residual Commons Value) — forward-looking; resource permanent-foreclosure; triangulated estimation; pairs with B2 (Foreclosure Bond).

Total Extraction Damages = CSD + uncaptured RCV. Per Block 1 Hole 5: you could fully compensate every Appalachian descendant — make them whole on health, education, housing, income — and the coal is still gone. Two distinct measurements of the same extraction event from different directions. Open Insight #19 (CSD-RCV correlation hypothesis) is the framework's deepest empirical claim — that CSD and uncaptured RCV correlate across extraction cases would be a structural empirical finding.

Four Gates (admission apparatus for Cᵢ)

Four conditions a candidate cost must pass to be admitted to RCV as a Cᵢ. Naming asymmetry honors structural distinction: Gate 1 (CIT) is a TEST (procedural methodology); Gates 2-4 are CONDITIONS (static mathematical properties).

  1. CIT (Commons Inversion Test) — content gate; is the cost commons-dependent? (Two sub-forms: Absence + Consumption Inversion.)
  2. Units Consistency — mathematical condition; cost in dollars × time-domain commensurable with rest of RCV integrand. (Renamed 2026-04-24 from "Dimensional Consistency" — per Four Gates cluster pass; "units" matches accounting-bookkeeping register, disambiguates from former "dimension" usage.)
  3. Boundedness — mathematical condition; cost integrand converges. (Active on climate/intergenerational cases requiring discount-rate convention.)
  4. Independence — mathematical condition; cost not redundantly correlated with already-admitted Cᵢ (no double-counting). (Active on multi-stream cases like 2008 financial crisis + opioid Appalachia.)

Commons Categories — 10 Examples (NOT canonical or exhaustive)

Per Option C' political-philosophical-accommodation discipline ratified 2026-04-24: the 10 categories below are EXAMPLES of commons CIT has surfaced across the 18 cases this book examines — not a canonical or exhaustive enumeration. Framework's universality is about METHOD (CIT applies anywhere extraction operates on commons-like resources), not about enumeration canonicity. Different political traditions have different commitments about what is shared and what is individual; framework accommodates rather than asserts.

"Abundance" preserved as STATE-name for each commons (abundant vs scarce — the state CIT inverts), distinct from the commons CATEGORY label. Per-commons rich-profile content lives in core/dimensions/commons_bonds_abundance_dimensions_v1_3_0.md (methodology doc; Phase A3 sweep target for v1.4.0 rename).

  1. Habitability — life-supporting conditions of place. Paradigm: McDowell coal extraction stripping clean air, healthy bodies, intact community life-support.
  2. Spatial — geographic shared infrastructure (mobility access; public space; geographic commons). Paradigm: Chesapeake Bay watermen's geographic commons.
  3. Temporal — shared time-budget; intergenerational time-commons. Paradigm: Social Security cohort-succession commons.
  4. Institutional — governance + legal architecture as shared infrastructure. Paradigm: 2008 financial-crisis institutional commons damage.
  5. Kindred — kinship + chosen-family + specific-attachment-bond infrastructure. Paradigm: opioid Appalachia family-destruction.
  6. Ecosystem — ecological services + biodiversity (canonical Ostrom commons territory). Paradigm: Deepwater Horizon Gulf ecosystem.
  7. Political — political agency + voice + representation as shared infrastructure. Paradigm: Political Capture Cost in extraction industries.
  8. Cohesion — civic / community / social-infrastructure binding (Klinenberg Palaces for the People territory).
  9. Epistemic — knowledge + epistemic-authority commons (Hess & Ostrom Understanding Knowledge as a Commons 2007 territory). Paradigm: opioid case truthful-info-suppression by Sackler/Purdue.
  10. Autonomy — autonomy-supporting infrastructure (Anderson Private Government 2017 — civic-republican framing). Most-contested example politically; see Open Insight #18 + Ch 6 political-philosophical acknowledgment passage.

This framework doesn't argue that autonomy is universally a commons, or that habitability is, or that time is. It argues that when any of these operate AS commons in a given society — shared, depended-on, collectively enabling — extraction can sever cost from that commons, and this methodology measures the severance. Different political traditions have different commitments. The framework works within any of those commitments. What it doesn't accept is extraction without accountability, regardless of which commons is being extracted from.

Retired Terms (preserved for provenance per Working Principle #4)

Abundance Inversion Test (AIT)
SUPERSEDED 2026-04-24 by Commons Inversion Test (CIT) per author insight that the test operates on commons (concrete shared resources), not abstract abundances. Decision-theoretic content preserved + refined via two CIT sub-forms; name updated to match operational instruction.
Asymmetric Regret Principle
SUPERSEDED 2026-04-24 by Asymmetric Regret Rule (ARR). Last-word rename: "Principle" overclaimed (operational decision rule, not foundational ethical principle). Decision-theoretic content unchanged; regret-theory framing preserved; "Rule" downgrades overclaim while preserving direct M6 academic pattern-match to Loomes-Sugden 1982 + Savage 1951.
Value Capture
RETIRED 2026-04-24. Ratified replacement: Value Extraction (per Value-Capture-vs-Extraction rigor pass + cross-pairing synthesis). Causal-chain bridge test favored Extraction (separation-from-source semantics pairs with Cost Severance's severance-from-capturer semantics).
Spatial Cost Severance (capitalized proper-noun) → spatial cost severance (lowercase prose phrase)
Capitalized form RETIRED → un-retired as lowercase prose phrase (per re-examination rigor pass 2026-04-24, reversing earlier Principle-#2-failure retirement). Geographic subtype of Cost Severance: costs concentrate where extraction happens; value disperses to distant consumers. Canonical case: McDowell County 13-year life-expectancy gap. Distinct from Spatial commons category (the abundance-domain). Lowercase discipline mirrors intergenerational cost severance pattern.
Temporal Cost Severance
RETIRED 2026-04-24 (capitalized form). Lowercase form also rejected per Temporal CS reconsider pass: Principle-#3 variant-subtype check FAILED for "temporal" — qualifier describes constitutive property of all CS (all severance has temporal offset between value-capture and cost-bearing), not a distinguishing gap-type. The intergenerational lowercase prose phrase covers the load-bearing cases. Spatial + intergenerational lowercase pair does NOT extend to a three-member cohort.
Abundance Dimension
RETIRED 2026-04-24 as construct-label. The 10 are now framed as commons categories (examples not canonical) per Option C' ratification + commons-as-structural-identity reframing. "Abundance" preserved as state-name (abundant vs scarce — the state CIT inverts), not as the construct-label for the 10.
Universality Test
RETIRED/DEMOTED 2026-04-24 per re-examination pass. Concept preserved in prose (scope claim + boundary-awareness illustration; Mars/Europa pedagogy retained in Tech Appendix) without dedicated capitalized term. Was 0 chapter refs; not exercised as test in book cases.
Full Generational Cost (FGC)
RETIRED 2026-04-24 (per tier-dissolution + meta-pass §11.1). FGC was tied to the 8-tier decomposition; under Path F + tier-dissolution ratifications, costs are now Cᵢ admitted via Four Gates. Where FGC was used, refer to RCV (the integrand) or Σᵢ Cᵢ (the sum of admitted costs).
Civilizational Substitutability Gap (CSG)
RETIRED 2026-04-24 per individual rigor pass + meta-pass §13.2.c. CSG = S_max(industrial) − S_max(existential) is a derived scalar from the Substitutability Function (S) primitive. Framework parsimony: don't name derivations of primitives. Concept preserved in prose as "industrial-vs-existential substitutability gap" + sub-entry under Substitutability Function in Tech Appendix.
Eight-Tier FGC Decomposition (v10)
RETIRED 2026-04-24 per tier-reframing rigor pass + Path F variable-addability ratification. The 8 tiers (Lifetime Survival Cost, Actuarial Risk Cost, Mission Failure Reserve, Dynastic Labor Cost, Foreclosure Cost, Community Transition Reserve, Ecological Carrying Cost, Knowledge and Cultural Cost, Political Capture Cost) failed the duplication check (Foreclosure Cost appeared in both tier 4 and as RCV core; the 8 couldn't be defended as non-overlapping/complete). REPLACED by Cᵢ indexed cost variables admitted through Four Gates. Some former tier names preserved as illustrative cost-category labels in chapter prose; no longer the framework's pre-sorted decomposition.
Cost Bearing
DEMOTED 2026-04-24 from glossary; retained in prose. Per author signal: "Cost Bearing is someone bearing the cost of cost severance, so perhaps prose vs. glossary." Role-description, not framework-technical term.
