When deciding acronym treatment in trade-press body prose, classify the acronym first:

**Pure-Mazzucato pattern (full-form everywhere; no acronym in body):**
- The full-form is plain-English self-explanatory ("value extraction", "cost severance", "moral limits")
- Each full-form use carries semantic content; substituting acronym would lose readability
- Examples: Mazzucato's "value extraction" 100+ times across *The Value of Everything* without compressing; Sandel's "moral limits" most occurrences in *What Money Can't Buy*

**Pure-Piketty pattern (introduce + deploy acronym):**
- The full-form is a less self-explanatory phrase OR functions as a named-ratio with mathematical definition (IPG = ratio; r > g = inequality; GDP = aggregate)
- Compact form serves prose-flow + table-header compactness
- Examples: Piketty's `r > g` 100+ times after one introduction; IPG in this manuscript (already disciplined)

**Sandel hybrid pattern (apply contextually, NOT mechanically):**
- The full-form is plain-English self-explanatory AND functions as a named quantity in equations
- Apply by-context: single-use occurrences expand to full-form; dense intra-paragraph repeat-references keep acronym; formulas/equations/named methods/table headers keep acronym
- Examples: RCV in this manuscript (Item 14 ratified 2026-05-11); Sandel's "the limits" / "the moral limits of markets" varying based on density

**Decision tree:**
1. Is the full-form plain-English self-explanatory? (RCV "residual commons value" yes; IPG "intergenerational pricing gap" less so)
2. Does the full-form function as a named ratio/quantity in equations? (yes for both)
3. If yes to #1 → Mazzucato or Sandel hybrid. If no to #1 → Piketty.
4. If yes to #2 AND yes to #1 → Sandel hybrid (the most nuanced case). Apply contextually.

**Why:** Surfaced 2026-05-11 during apparatus register sweep Item 14 (RCV acronym treatment). Initial recommendation was Piketty pattern (introduce + deploy); pressure-test from author led to discovering RCV is hybrid territory. Substitution test revealed: single-use RCV expands cleanly to "residual commons value" with semantic gain; dense intra-paragraph repeat (e.g., Ch 6 line 615 area: 3 RCV uses in consecutive sentences after full-name introduction) keeps acronym for prose-flow. Sandel does this naturally with "moral limits" — the trade-comp anchor.

**How to apply:** When judging acronym treatment, run the 4-step decision tree above. For Sandel-hybrid cases:
- Default expand to full-form for body prose
- Keep acronym in formulas, equations, named methods (e.g., "Method 1"), table headers
- Keep acronym in dense intra-paragraph repeat-reference where 2+ uses appear within a paragraph after full-name introduction
- Keep acronym in parallel-structure case-walks where multiple chapters or paragraphs use the same compact reference for cross-case comparison
