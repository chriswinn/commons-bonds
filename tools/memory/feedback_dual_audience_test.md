When polishing apparatus, acronyms, equations, cross-references, or any apparatus-bearing prose for trade-press, test each rewrite against BOTH audiences explicitly:

- **Layman/non-specialist** (e.g., a Mazzucato/Raworth-cluster intellectual reader without economics PhD training) — does the rewrite give them a foothold? Are technical terms either plain-English-readable or scaffolded with gloss? Does scannable structure (named navigation aids, full-form names, descriptive cross-references) work for them?
- **Specialist** (e.g., Coasean economist, Berggruen Prize judge, academic reviewer) — does the rewrite preserve precision? Are formal cross-references intact? Does the apparatus still earn its place where it lives?

**Why:** Surfaced 2026-05-11 during the apparatus register sweep — author caught me defaulting to "shorter = trade-press-friendly" on the Hotelling Identity passage (Item 8). Original recommendation compressed 5 sentences to 2 with semicolons; substance was dense; a layman ("somewhat of a layman of economics") would have been over-loaded. Right answer was Option C polish-not-compress: address weak sentences (Pigou circularity), drop jargon compounds ("externality-on-cost-bearers" → "ongoing harm-to-others"), merge redundant pair — same length, better dual-audience landing. Subsequent items (9, 10, 11, 14, 15) all benefited from this discipline.

**How to apply:** When judging an apparatus/register decision, BEFORE applying the edit:
1. Name the layman version of the audience (which trade-press reader is at the entry-end of the comp shelf?)
2. Name the specialist version of the audience (which prize-track or academic reader needs precision?)
3. Substitute the proposed rewrite. Read it through both audiences.
4. If only one lands cleanly, look for a third option that serves both. Often: named navigation aid + descriptive content + brief plain-English foothold.
5. If "shorter" is the default tendency, pause and ask "what does the substance require?" Per `feedback_substance_drives_length.md` — substance drives length, not the other way around.

**Trade-comp test:** Mazzucato (*Value of Everything*), Raworth (*Doughnut Economics*), Sandel (*What Money Can't Buy*), Pistor (*Code of Capital*), Sen-in-trade-mode (*Development as Freedom*) — none of them defaults to compression-as-virtue. Each varies prose density based on substance density. Default to that pattern.
