# Peer-review defense for posited numbers in the quantitative apparatus (ratified 2026-06-09)

**The discipline.** Treat every number in the framework's quantitative apparatus — the Technical Appendix above all — as something a referee will defend-or-attack. Four moves convert a contested/posited number from a liability into a defensible (often a robustness-demonstrating) cell:

1. **Source-or-label — never a third option.** Each number is either (a) cited to an external source, or (b) explicitly labeled as the authors' own estimate/assumption with the derivation shown. The fatal case is (c): a posited number typeset to look sourced. A referee who catches one unsourced figure dressed as sourced stops trusting *every* number in the table — including the well-sourced load-bearing ones.

2. **Show insensitivity — don't assert it.** For any contested input, add an explicit sensitivity statement demonstrating the conclusion does not depend on it (e.g., "the anchor is dominated ~50× by the sourced X term; component Y is <4% and the result is robust to any plausible value of it"). This converts the weakest cell into a robustness result. Immateriality *asserted* is worse than useless ("then why is it dressed up with precise numbers?"); immateriality *demonstrated* is a referee-rewarded defense.

3. **Argue conservatively (a fortiori).** Where a documented figure is lower than the framework's estimate, anchor on the documented (sourced) one and state plainly that the true value is higher, which only strengthens the claim. A referee cannot kill a number that is sourced, explicitly conservative, and makes your case anyway. Claiming the high posited number is attackable; claiming the low sourced number and noting the truth exceeds it is not.

4. **Route the "worth more" claim to the method built for it; keep methods un-conflated.** In this framework the claim "the true cost exceeds any substitution proxy" belongs in Method 3 (scarcity-adjusted option value / irreversibility premium), not as a posited Method-1 dollar figure. And never feed a Method-2 quantity (realized spend / bonds-posted) into Method 1 (replacement cost) — that conflation collapses the accountability gap (RCV − B) by construction, and a sharp referee catches it instantly. NB the distinction between a cleanup-*cost* estimate (can inform M1) and bonds-*posted* (is M2).

**Path A vs Path B sequencing.** The non-numerical hardening — the sensitivity statement + labeling + M3 routing ("Path A") — is safe to apply anytime and is usually *sufficient* for defense. The actual number re-anchor ("Path B") often ripples into downstream ratios (RCV / IPG / CS) owned by other sessions, so hand it to the owning recompute rather than doing it in isolation; once Path A shows insensitivity, Path B is optional.

**Why:** An unsourced number that looks sourced is the single most damaging thing in a results table; the damage is reputational and propagates (the referee distrusts the whole apparatus, not just the one cell). The goal is not to find the "right" number but to remove the attack surface and demonstrate robustness. Same root discipline as the no-invented-sources hard rule [[feedback_no_invented_factual_claims_in_publisher_facing_prose]] — cite the external source, never the framework's own posited / "canonical" status.

**How to apply:** When auditing or editing any quantitative TA/chapter passage: (1) flag every number as sourced / labeled-estimate / framework-output; (2) for any posited or contested input, add or confirm a sensitivity statement showing the headline is insensitive to it; (3) prefer the lower documented anchor + a-fortiori framing over a higher posited one; (4) push "the real cost is larger" into Method 3, not a Method-1 figure; (5) if the number change touches downstream ratios another session owns, hand it off (Path B) rather than thrashing.

**Empirical anchor:** §11.6 McDowell Method-1 Ecosystem floor, 2026-06-09. The posited $50,000–200,000/acre full-restoration estimate was unsourced and sat typeset like the sourced Habitability (DAC) term beside it. Resolution: labeled it an estimate, cited the documented conservative anchor ($12,000–15,500/acre; Appalachian Voices 2021) for comparison, added a Method-1 **Sensitivity** statement (Eco+Cohesion <4% of the combined anchor, dominated by the sourced Habitability term, result robust), and routed the full-restoration "worth more" claim to the Method-3 irreversibility premium — all with **no headline number change** (Path A). The number re-anchor (Path B) went to the IPG/M3 rework sessions as optional. The author framed the trigger question as: "think of this from the point of an academic paper about to be peer-reviewed and defended."
