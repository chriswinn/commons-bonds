# feedback_plain_figure_register_over_qualified_shock

**Ratified:** 2026-06-11 (Ch 3 Tangier erosion figure, Option B reverted to Option A same day)

When a vivid or shocking figure is only honest with attribution-and-qualification apparatus wrapped around it (e.g. "the figure that travels in the press, fifteen feet of shoreline a year, describes the worst of the exposed western shore; the peer-reviewed average is closer to two feet"), the author prefers the plain sourced figure stated flatly, with no apparatus. Author, redirecting the Ch 3 Tangier passage from the ratified dual-figure reframe (Option B) to the paper-average-only correction (Option A):

> "Stick with Option A versus trying to reach for shocking rhetoric that needs a lot of qualifications to be true. We have chosen a different path than that throughout this entire book."

The same principle retires unsourced flourishes that overclaim: "the average never takes a year off" was dropped in the same edit because it asserts year-by-year monotonicity the source's interval record does not literally establish.

**Why:** The book's register is plain accounting, not maximal impact. Qualification scaffolding around a big number signals reaching; a smaller true number stated flatly carries more authority in this voice and is peer-review-defensible without footwork. This is the prose-register sibling of the conservative-argument rule in `feedback_quantitative_apparatus_peer_review_defense.md` ("anchor on the lower sourced figure, note the truth is stronger").

**How to apply:** In rigor-pass options for figure corrections in book chapters, still PRESENT the keep-the-vivid-figure-with-attribution option (it is honest and sometimes right for magazine registers), but default the RECOMMENDATION to the plain paper-figure option. Audit new prose for figures that need a dependent clause to be true — each is a candidate for the plain substitute. Derivative essays get their own call at their own ratification, but the default has moved (empirical anchor: the Atlantic-main redraft candidates A + B both carried the dual-figure construction for the same Tangier passage and were flagged to that workstream 2026-06-11).
