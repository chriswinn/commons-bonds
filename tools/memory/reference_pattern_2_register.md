# Pattern 2 register for external-facing framework material

**Where to find canonical analysis:** `core/methodology/decision_time_application_internal_v1.0.0.md` §3 — preserves the literature-pattern audit (Pattern 1 / 2 / 3) that grounds this register choice.

## Three structural patterns observed in comparable framework books

| Pattern | Exemplars | Decision-time/applied treatment | Fit for Commons Bonds |
|---|---|---|---|
| **Pattern 1 — FOREGROUNDED** | *Nudge* (Thaler-Sunstein 2008); *Predictably Irrational* (Ariely 2008) | Dedicated how-to chapters + applied weight | **NO** — strains thick framework + author shield mismatch + Insight #13 register |
| **Pattern 2 — THREADED** | *Doughnut Economics* (Raworth 2017); *Mission Economy* + *Value of Everything* (Mazzucato 2018, 2021); *Mine!* + *Gridlock Economy* (Heller 2008, 2021); *Thinking Fast and Slow* (Kahneman 2011) | Case demonstration with decision-time framing in chapter walkthroughs; no methodology codification | **YES** — fits framework thickness + author shield + audience choice (Insight #25 academic-trade hybrid) + epistemic-humility (Insight #14) + book-scope discipline (Insight #13) |
| **Pattern 3 — EX-POST ONLY** | *Capital in the 21st Century* (Piketty 2014); *Governing the Commons* (Ostrom 1990); *Great Transformation* (Polanyi 1944); *Debt* (Graeber 2011) | Diagnostic-heavy; light or no decision-time tooling | **NO** — underclaims practical reach for chosen audience |

## Why Pattern 2 wins for Commons Bonds

- **Trust the reader to extract the methodology.** Readers see the framework get used across cases; they internalize how to use it from observation — the book doesn't codify a methodology, which prevents register-mismatch with framework-naming Book 1 vs applied-advocacy Book 2.
- **Native register-fit.** Framework + chapter-walkthrough format reinforces "this is how the framework illuminates this domain" — naturally extends to "this is how the framework can illuminate your decisions."
- **Compatible with epistemic-humility discipline (Insight #14).** Same register as Norway/Accountability Bond treatment: *"here's what the framework reveals; here's what's been tried; here's what's working in practice."*
- **Institutional shield matches book scope.** Raworth at Oxford/Cambridge; Mazzucato at UCL; Heller at Columbia Law — institutional but not coauthored-advocacy. Same shield class Commons Bonds has via being-source-of-framework + Berggruen/Noema track + nursing career.

## When to apply

**External-facing material that touches applied / decision-time / how-to registers:**
- Chapter prose at any layer (drafts, GuidanceDocs that feed drafts, supplementary that integrates)
- Glossary entries (definitions only; not how-to)
- Tech Appendix prose (formal apparatus + cases; not codified methodology)
- Essay material (Noema / Berggruen submissions)

**Pattern 2 discipline:**
- Demonstrate the framework's affordance through case treatment (chapters; personal stories; cross-references)
- Do NOT codify a methodology in publisher-facing prose (no "How to Apply" chapters; no prescribed two-step procedures; no "Five Steps to Use This Framework")
- Trust the reader to extract the methodology by observing the framework get used
- Mirror the Insight #14 epistemic-humility discipline: *"demonstrate the affordance; don't codify it"*

## When NOT to apply (i.e., these go internal)

- Codified methodology (e.g., the informal CIT + informal RCV two-step) — internal scaffolding
- Worked decision-time examples (apartment leases, employment offers, medical-treatment decisions, career pivots, etc.) — internal scaffolding (some may migrate to chapter prose if they earn the slot via Pattern 2 demonstration)
- Literature audits like this one — internal scaffolding
- Book 2 seed material — internal scaffolding
