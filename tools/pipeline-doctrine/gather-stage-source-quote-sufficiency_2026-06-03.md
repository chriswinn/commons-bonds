# Source/quote sufficiency belongs in GATHER

**The catch (author, 2026-06-03):** a "quote-enrichment pass" should never be a separate downstream stage. It is part of Stage 0 / gather — "look at what we have, then see if something better exists." Offering enrichment as a late pass is a tell that the gather under-specified quote-sourcing.

## The fix
Amendment E's front-end gather is three layers (collect-what-exists / field-map → goal-directed overlay → outstanding-needs note). Add an explicit **source/quote-sufficiency sub-step** inside it:

1. **Inventory** the citable quotes/sources available for each load-bearing beat (from the corpus, `research/`, and — where the venue rewards direct quotation — targeted public-source search).
2. **Test for "something better."** For each beat, ask whether a stronger, more exact, or more current *citable* quote exists than the one currently in hand. This is the gather's defining move applied to quotes.
3. **Select the best**, with provenance (two-state citation model: tentative-with-provenance → confirmed-on-landing; fabrication gate at harvest — no quote enters without a real, attributable source).
4. **Flag gaps in the outstanding-needs note.** A beat carried by a weak paraphrase where a strong citable quote would do better is an outstanding need, resolved BEFORE drafting — not after.

Consequence: there is no "enrichment pass," because gather already did it. If a drafted artifact looks under-quoted, the diagnosis is "its gather was incomplete," and the remedy is to *complete the gather* (same stage, up front), not to bolt on a downstream pass.

## Provenance / citability discipline (carried from named-subject-consent + no-fabrication)
- Public-record / on-record quotes are citable by default (courtesy-notify, not consent).
- Living-private subjects: anonymize unless consent; never mine private correspondence/interview prep for attributed quotes.
- Every quote harvested at gather carries its source; nothing enters a draft without an attributable, verifiable source (fabrication gate).

## Empirical note (the essays in flight)
The originals' gathers largely already did this: Atlantic Main selected 16 vetted public-record quotes from the ~30+ quotable subjects the post-petition research surfaced; NYRB pulled its book quotes; $100 Barrel / FA gathered the named-thinker + case sources. The fresh drafts preserved those, so the best-available quotes were already in hand — confirming that when gather is done well, no separate enrichment is needed. The lesson is to NAME quote-sufficiency as a gather sub-step so it is never again treated as an afterthought.

## Applies to chapters too
Per the chapter-pipeline thinking (`chapter-pipeline-adaptation-THINKING_2026-06-03.md`): a chapter's sources/quotes get gathered best-available at the front-end, with the cross-chapter coherence spine telling gather which sources a chapter must carry (and which it inherits from sibling chapters).
