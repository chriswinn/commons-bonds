### §4.2 Chapter-level chain readability

Sample 3-minute explanation ("dinner-table test"):

> *"The framework is called Commons Bonds — about the shared things we depend on and the bonds that should hold people accountable for what they do to those shared things. When someone extracts value from a commons — that's Value Extraction — they leave costs behind. We call that Cost Severance: the structure that lets the value-capturer walk away while the community, ecosystem, or future generations pay the bill. To figure out which costs actually count, we use the Abundance Inversion Test — AIT — which asks whether each candidate cost only shows up when the underlying resource is scarce. If yes, it counts; we add it as one of the costs Cᵢ. All the costs that count add up — weighted by time and probability — to the Residual Commons Value, RCV. That's the true cost. If the extractor paid B through accountability bonds, the difference RCV minus B is the Severed Cost — the bill the commons bear."*

**3 minutes; 7 technical terms introduced cleanly; each term has a specific role; listener parses each position; end is a quantifiable claim.** Chain flows.
