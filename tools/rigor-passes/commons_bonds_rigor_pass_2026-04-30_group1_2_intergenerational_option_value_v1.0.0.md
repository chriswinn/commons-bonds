## §4. Working definition + framework usage

### §4.1 Working definition

**Intergenerational Option Value** = the value of preserving optionality on a resource for future-generation uses that may emerge from future technologies / discoveries / civilizational-application changes. Specialization of real-options option-value concept to intergenerational temporal-scope.

### §4.2 Framework usage at Ch 4

Ch 4 *The Existence Proof* uses option-value reasoning to establish the argumentative core: future generations have value-claims on present extraction *because* current extraction forecloses option-value they would otherwise hold. The "existence proof" is that this option-value is real (non-zero) under any reasonable preservation-of-future-utility assumption.

If Intergenerational Option Value is removed from the chapter's apparatus, the existence-proof argument needs reformulation via either (a) substitutability-function-only framing (S(t) approaches 0 as foreclosure progresses), OR (b) externality-tail-only framing (E(R, t) accumulates intergenerationally). Either reformulation loses precision — substitutability captures *probability* of substitute existing, not *value* of preserving the option to discover one; externality tail captures *cost* of remediation, not *value* of preserved optionality.

**Conclusion:** Intergenerational Option Value names a distinct concept that Ch 4's argument-spine depends on.
