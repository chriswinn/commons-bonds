## §1. Why this pass matters now

Pass 3.2 is the voice-polish pass — the polarity-CUT pass that chisels LLM-tics, em-dash crutches, meta-commentary, throat-clearing, and declarative-three-in-a-row LLM cadence out of the post-Pass-3.1 prose. The Atlantic features editorial register is **cumulative-portrait reportage** — the argument emerges from the voices, not from the writer narrating his way through it. The audit bar is the Atlantic features editor's first-3-second / first-paragraph / first-section filter: does this open like an Atlantic feature, hold like one across the cumulative-portrait architecture, and close like one rather than like an Atlantic Ideas declarative-thesis essay or a third-person argument-and-defend op-ed.

**The dominant Atlantic-features-register drift pattern** for this essay (per brief §2): **Atlantic Ideas register seepage** — declarative-thesis-and-defend cadence (the register the Atlantic Ideas sub-editor pool wants) seeping into the cumulative-portrait register (the register the Atlantic features sub-editor pool wants). Specific markers: thesis-state sentences in observational positions; "what the framework shows..." / "the structural observation is..." analytical-narrator interjections; first-person plural "we" intrusions that shift the voice from observational-witness to editorial-essayist; explicit self-narration about the essay itself ("I have spent a good deal of this essay on..." / "I want to be careful here about a register..." / "What I want the reader to hold...").

**The em-dash discipline** ([`feedback_em_dash_overuse.md`](../memory/feedback_em_dash_overuse.md) 2026-05-21 RATIFIED) is the second dominant calibration: **treat em-dash use as a flag requiring active justification; default to commas / periods / restructure for smoother prose.** Cumulative-essay em-dash count is itself a voice-polish concern. Per chapter-level Ch 3 Pass 3.2 predecessor (2026-05-20), the same em-dash discipline cut Ch 3 chapter em-dashes significantly; the audience-blind Stage 2 essay regenerated fresh prose without inheriting that discipline pre-Pass-3.2.

**The polarity discipline is strict:** Pass 3.2 cuts; Pass 3.5 (later session) restores. If a cut feels like it loses substance, flag for Pass 3.5 consideration rather than holding the cut. Restoration is Pass 3.5's polarity, not Pass 3.2's.
