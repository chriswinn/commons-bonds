## §2. The question under test

### §2.1 The phenomenon (per v2 glossary)

> *"Costs concentrate where extraction happens; value disperses to distant consumers. Geographic dimension of cost severance. The 13-year life expectancy gap between McDowell County and the national average is the human face of spatial cost severance: coal's value dispersed nationally while health costs and community collapse concentrated locally."*

This describes a real phenomenon: geographic separation between value-capturers (distant consumers of coal electricity) and cost-bearers (local communities).

### §2.2 Why this pass re-examines the earlier retirement

Earlier retirement ratification (commit `2dc54aa`, 2026-04-24) was based on Chris's articulation 2026-04-24:

> *"Spatial Cost severance is probably misnamed, as that was just Cost Severance that happened only if and when distance between locations vanished. So that isn't really cost severance but really an abundance cost that vanishes or appears e.g. miner can teleport instantly to/from an asteroid for work each day = totally removes a lot of the cost severance of living on an asteroid."*

**The conflation:** Chris's asteroid-teleport example describes SPATIAL-ABUNDANCE MECHANICS (when distance is an abundance condition, certain costs vanish when distance collapses). The v2 glossary definition describes GEOGRAPHIC DISPERSAL OF VALUE vs CONCENTRATION OF COST — a different phenomenon (value goes to distant consumers; costs stay at extraction site; severance operates across the geographic gap).

Both are real. The asteroid-teleport describes abundance-cost mechanics (Spatial abundance). The v2 McDowell description describes cost-severance-across-geographic-gap. **They're distinct phenomena that got conflated** in the earlier retirement reasoning.

**B — Adopt "spatial cost severance" as lowercase prose phrase.** Analogous to the Intergenerational Cost Severance 2026-04-24 adoption. Natural composition; not a proper-noun compound.

**C — No dedicated phrase; describe in prose.** Each chapter describes the phenomenon in its own words (e.g., "coal's value dispersed nationally while health costs concentrated locally") without a framework-specific phrase.

**D — Rename to something different** (e.g., "geographic cost severance," "cross-spatial severance," "dispersal severance").

**E — Retire entirely** (current ratification, which this pass reconsiders).

**F — Placeholder** for anything else surfacing.

## §3. Principle #3 (misnaming) re-examination

**Corrected test (v2 definition basis):**

The phenomenon per v2: *"Costs concentrate where extraction happens; value disperses to distant consumers."*

**Counterfactual:** remove the spatial dispersal of value. What happens?

If all value from extraction stays in the local geography (local community captures value + bears cost), there's no separation between value-capturer and cost-bearer — they're the same people. Cost Severance mechanism can't operate without the separation.

**So:** spatial dispersal is LOAD-BEARING for this form of severance. Removing it eliminates the severance. This matches Principle #3's subtype-verification test:
- Remove the condition → severance vanishes → condition produces a genuine subtype.

**Contrast with Chris's earlier asteroid-teleport reasoning:** the teleport scenario had miners/community in the same location (teleport collapses distance). In that case, cost severance might still operate (value going to Earth-based shareholders; cost-bearing on Mars) — it just wouldn't be SPATIAL severance; other severance dimensions (intergenerational, political) might still apply.

**Principle #3 verdict:** PASSES. Spatial Cost Severance is a genuine subtype of Cost Severance. Earlier retirement based on misnaming was INCORRECT.

## §4. Option B — the lowercase prose phrase adoption (detailed)

### §4.1 Composition

"spatial cost severance" composes naturally:
- "spatial" — framework-familiar adjective (currently in use with abundances — "Spatial abundance")
- "cost severance" — Ring-1 flagship mechanism name

Composition mirrors the Intergenerational Cost Severance pattern that was adopted 2026-04-24.

### §4.2 Lowercase discipline

Same discipline as intergenerational cost severance:
- Lowercase always; restructure sentences to avoid sentence-start capitalization where possible.
- Not a capitalized proper-noun compound — composes two existing framework concepts.
- Style discipline prevents drift back toward jargon.

### §4.3 Why it works for the book's rhetorical strategy

- Accounting register: "spatial cost severance" reads as ordinary-language description of how value disperses while costs concentrate.
- Specificity preserved: each case describes the specific geographic pattern (McDowell: coal's national value vs local concentration; Chesapeake watermen: Bay's distant markets vs local ecosystem; Libby vermiculite: distributed products vs concentrated asbestos exposure).
- Subtype-naming capacity: when cross-case comparison calls for distinguishing "severance-across-space" from "severance-across-generations" from "severance-across-institutions," the lowercase phrase provides a clean handle.

### §4.4 Contrast with intergenerational cost severance

Both adopted as lowercase prose phrases, both Principle-#3-passing, both following the same pattern. The two together:
- "intergenerational cost severance" — severance across generational gap
- "spatial cost severance" — severance across geographic gap

Could be joined by future subtype descriptors (e.g., "institutional cost severance" — severance across institutional layers) if framework development surfaces analogous patterns. All composed naturally; no jargon inflation.
