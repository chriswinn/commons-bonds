## Evaluation dimensions

For each load-bearing element of the pre-pass-1 essay, evaluate:

- **Editor function** — advances Noema's three-register weave? Satisfies named-sources + concrete-numbers expectations? Within internationalist scope?
- **Reader resonance** — lands with Mazzucato/Raworth-cluster intellectual?
- **Agent function** — book-deal pipeline strength signals (voice, authority anchors, comp-titles signaling, blurb potential, narrative spine)?
- **Publisher function** — trade-press commercial viability (category, hook, accessibility, review-network)?
- **Berggruen-track function** — points toward prize-track register or away from it?
- **Cultural-engagement check** — handles lineage / appropriation concerns?
- **Apparatus check** — introduces jargon / formal apparatus that hurts Noema register?
- **Duplicate-weight audit** — same point carried by another passage; which carries it better, for which audiences?

**Verdict format:** **Lock** / **Keep** / **Compress** / **Cut** / **Fix**

Symbol shorthand in pressure-test breadth: ✓✓✓ (essential) / ✓✓ (strong) / ✓ (helps) / ⚠ (mild concern) / ⚠⚠ (real concern) / ⚠⚠⚠ (disqualifying) / – (not load-bearing for that audience).
