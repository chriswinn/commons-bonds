## §1. Executive summary

**RECOMMENDED: Option A — KEEP RETIRED. Do NOT adopt "temporal cost severance" as a lowercase prose phrase.** The retirement of the capitalized "Temporal Cost Severance" (commit `4435488`) stands; the lowercase form should also stay retired. "Intergenerational cost severance" (adopted 2026-04-24 as lowercase prose phrase) covers the load-bearing cases; adding "temporal cost severance" as a parallel lowercase phrase fails Principle #3's variant-subtype check.

**The decisive argument:** Cost Severance inherently has a temporal element — costs bear after value is extracted, almost by definition. "Temporal" is therefore a property of ALL severance, not a distinguishing subtype. Contrast with *spatial* and *intergenerational*:

| Subtype qualifier | Principle-#3 counterfactual | Distinguishing? |
|---|---|---|
| **spatial** cost severance | Remove geographic gap → severance vanishes (value-capturers + cost-bearers become same community) | YES — distinct subtype |
| **intergenerational** cost severance | Remove generational gap → severance reduces to within-lifetime forms | YES — distinct subtype |
| **temporal** cost severance | Remove temporal gap → value and cost are simultaneous → but *all* severance has some temporal offset | NO — property of all CS, not a subtype |

**So:** the spatial-intergenerational pair are both narrow gap-type qualifiers that pass Principle #3 because removing the specific gap eliminates the specific severance. "Temporal" is a broader qualifier that describes a property of every severance instance; removing temporality doesn't identify a distinct subtype — it abolishes severance entirely.

**What this means:**
- The spatial / intergenerational lowercase pair stops at two. No "temporal" third sibling.
- If framework development surfaces OTHER specific gap-types (institutional, political, demographic) that pass Principle #3, they can be added as lowercase phrases. "Temporal" doesn't.
- "Temporal Cost Severance" (capitalized) stays retired — the original retirement verdict holds.

## §2. The question under test

### §2.1 Why this pass exists

The Spatial Cost Severance re-examination (commit `c04b7d8`, 2026-04-24) reversed the earlier retirement and adopted "spatial cost severance" as lowercase prose phrase. The pass author (Chris Winn) asked whether Temporal Cost Severance should parallel — i.e., also be un-retired as lowercase prose phrase — creating a three-member sibling cohort: spatial / intergenerational / temporal cost severance.

### §2.2 Options

- **A — KEEP RETIRED.** Capitalized "Temporal Cost Severance" stays retired (per commit `4435488`); no lowercase prose phrase adopted. The concept is carried by "intergenerational cost severance" for the load-bearing cases.
- **B — ADOPT "temporal cost severance" as lowercase prose phrase,** parallel to spatial + intergenerational, forming a three-member cohort.
- **C — Rename "intergenerational cost severance" to "temporal cost severance"** (treat temporal as the genus; lose intergenerational altogether).
- **D — Introduce "temporal cost severance" as a genus phrase with "intergenerational cost severance" as a species** (two-level hierarchy).

### §2.3 Sub-claims

**Option A (KEEP RETIRED):**
- SC-A1: Cost Severance inherently has a temporal element; "temporal" is not a distinguishing subtype qualifier.
- SC-A2: Principle #3's counterfactual (remove the temporal gap → severance vanishes) fails because removing temporality abolishes severance entirely, not a specific subtype.
- SC-A3: Intergenerational cost severance already covers the load-bearing cases (generational-gap severance).
- SC-A4: Vocabulary footprint discipline — adding a third sibling that doesn't pass Principle #3 is jargon inflation.

**Option B (ADOPT):**
- SC-B1: Symmetric-cohort appeal (spatial + intergenerational + temporal forms a clean triad).
- SC-B2: Decade-scale temporal severance (non-generational) might need its own phrase (e.g., deferred-liability pattern: Q4 value, Q8 cost bearing).
- SC-B3: Future framework cases might distinguish temporal from intergenerational.

**Option D (GENUS-SPECIES):**
- SC-D1: Temporal as genus; intergenerational as species.
- SC-D2: Hierarchical precision serves academic-audience uptake.

### §2.4 Falsifiers

**Option A is falsified if:**
- A load-bearing case exists where "temporal cost severance" is needed and "intergenerational cost severance" fails to carry.
- The three-member symmetric cohort delivers adoption-anchoring benefit exceeding the footprint cost.

**Option B is falsified if:**
- Principle #3 check fails (temporal-gap removal doesn't identify a distinct subtype).
- The phrase duplicates the work intergenerational cost severance already does without adding precision.

## §3. Principle #3 variant-subtype test (decisive module)

### §3.1 Test framework

For a subtype qualifier Q to pass Principle #3 as a distinct Cost Severance subtype:
1. Remove the specific condition that Q describes.
2. Does severance in the phenomenon vanish?
3. If YES: Q identifies a genuine subtype.
4. If NO: Q is either not distinguishing (too broad) or not load-bearing (too narrow).

### §3.2 Running the test on *spatial*

Phenomenon: McDowell County coal extraction — value disperses nationally; costs (health, community) concentrate locally.
Remove the spatial gap: imagine all consumers of electricity live near McDowell County.
Result: value-capturers and cost-bearers are the same community. Severance mechanism can't operate across a collapsed geography.
**Verdict: PASSES — spatial gap is load-bearing for this severance subtype.**

### §3.3 Running the test on *intergenerational*

Phenomenon: Social Security pay-as-you-go + underfunded pensions — current generation books retirement benefits; future generations bear the fiscal burden.
Remove the intergenerational gap: imagine all beneficiaries and all cost-bearers are within a single generational cohort.
Result: intra-generational deferred-liability pattern — still severance, but a different subtype (e.g., life-cycle deferral). The intergenerational-specific phenomenon disappears.
**Verdict: PASSES — intergenerational gap is load-bearing for this severance subtype.**

### §3.4 Running the test on *temporal*

Phenomenon: ANY cost severance involves a temporal offset — costs always bear after value is extracted (if only microseconds). There is no "simultaneous-value-and-cost severance."
Remove the temporal offset: value and cost are perfectly simultaneous.
Result: severance doesn't exist at all. There's no subtype that disappears; severance itself disappears.

**Verdict: FAILS as variant-subtype qualifier.** "Temporal" describes a property of all severance (temporal offset is constitutive), not a specific subtype distinguishable from other severance forms.

### §3.5 Why the spatial-intergenerational pair isn't a pattern that extends

The spatial / intergenerational pair both describe **specific gap-types** along which severance operates (geographic distance · generational cohort). Each specific gap-type can be present or absent; when absent, *that specific subtype* disappears while other severance forms persist.

"Temporal" is not a specific gap-type. It's the *general property* that there's an offset between value-capture and cost-bearing. This property is always present in severance; its absence doesn't identify a missing subtype — it identifies the absence of severance itself.

**Diagnostic:** if removing a qualifier's condition doesn't leave other severance forms intact, the qualifier isn't a subtype qualifier.

## §4. Options D (genus-species) consideration

Option D would treat "temporal cost severance" as a genus phrase under which "intergenerational cost severance" sits as a species. Analogous to "mammal → primate."

**Problem:** The genus isn't a meaningful category because it encompasses ALL cost severance (all severance is temporal). A genus that equals its parent class isn't a useful hierarchical layer.

Compare:
- "temporal cost severance" (proposed genus) = "cost severance" (existing parent) — identical extension.
- "intergenerational cost severance" (species) ⊂ "cost severance" — proper subset.

The genus collapses to the parent. Option D introduces a redundant layer without gaining specificity.

**Verdict:** Option D rejected on category-theory grounds.

---

## §5. Option C consideration (rename intergenerational → temporal)

Option C would replace "intergenerational cost severance" with "temporal cost severance."

**Problem:** This loses precision. "Intergenerational" specifies a particular generational-cohort gap; "temporal" does not. The framework's heavy use of "intergenerational" (109 occurrences) + "future generations" (72 occurrences) demonstrates the specific qualifier is load-bearing. Downgrading to "temporal" would fuzzify.

**Verdict:** Option C rejected on precision grounds.

## §6. M1–M11 abbreviated results

- **M1 CORE:** Indifferent. Framework math unchanged either way.
- **M2 Case study:** Intergenerational cases (Social Security, carbon, climate debt) all carry under "intergenerational cost severance"; adding "temporal" adds nothing.
- **M3 Book content:** Zero friction keeping retired (no rewrite); ~5 references would need to be added if Option B adopted.
- **M4 Craft:** "Temporal" is jargon-register without precision gain; "intergenerational" is accounting-register + specific.
- **M5 Dinner-table:** Both phrases parse; intergenerational is more evocative ("across generations") than temporal ("across time — which time?").
- **M6 Academic:** Public-finance + environmental-ethics literature uses "intergenerational" heavily; "temporal cost severance" would create unnecessary synonym noise.
- **M7 Originality:** Unchanged.
- **M8 Long-term:** Keep-retired wins — lower footprint, clearer adoption anchor (intergenerational).
- **M9 Risk:** Adopting temporal re-introduces the retirement-then-un-retirement whiplash without a Principle-#3 justification.
- **M10 Publishing:** Neutral.
- **M11 Critic:** "Why keep 'temporal' when 'intergenerational' does the work and is more precise?" — unanswerable under Option B. Option A survives.

**§22.4 alignment:** Option A positive (footprint discipline + goal-aligned precision). Option B neutral-to-negative.
