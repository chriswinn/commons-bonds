## §2. The question under test

### §2.1 The decomposition (per Three Ways + RCV Formal-Model rigor pass §7.3)

The framework's two-instrument architecture (Block 1) distinguishes:
- **CSD (Cost Severance Damages):** backward-looking realized human harm.
- **RCV (Residual Commons Value):** forward-looking resource permanent-foreclosure.

The accountability-instrument side of the framework therefore decomposes:
- **B1 — Reparations Bond:** the instrument(s) that close the CSD gap. Backward-looking; reparations-economics-grounded; welfare-state + reparations-payment + environmental-justice-damages compensation instruments.
- **B2 — Resource-Foreclosure Bond:** the instrument(s) that close the RCV gap. Forward-looking; Hartwick-savings + sovereign-wealth-fund + EIB + reclamation-bond instruments.

Aggregate B = B1 + B2.

CS equation refined:
- **Total CS = (CSD − B1) + (RCV − B2)** = (CSD + RCV) − (B1 + B2) = Total Extraction Damages − Total Accountability Bond.

### §2.2 Why this matters

**Without decomposition:** "Accountability Bond" is single aggregate quantity; analysis is at coarse-grained level.

**With decomposition:**
- Norway's SWF cleanly identified as B2-canonical exemplar (operationalizes Hartwick rule).
- Reparations-economics + welfare-state architecture cleanly identified as B1 territory.
- Distinct policy-design questions for each (B1 instruments operate post-extraction with realized-damage measurement; B2 instruments operate during/after extraction with forward-looking foreclosure pricing).
- Different real-world institutions specialize in each (Norway SWF for B2; civil-rights litigation + welfare-state programs for B1).

### §2.3 Options

As in §1.

### §2.4 Sub-claims for Option A

- SC-A1: Two-instrument architecture (Block 1) implies B decomposition; making it explicit aligns accountability-instrument layer with measurement layer.
- SC-A2: Equational form preserved (CS = RCV − B with B = B1 + B2).

## §3. Equational consistency check (decisive module)

### §3.1 Aggregate equation

Currently ratified: **CS = RCV − B** (Ring-2; flagship-equation; commit `220914f`).

### §3.2 Two-instrument decomposition

Per Block 1: Total Extraction Damages = CSD + uncaptured RCV.

If we decompose B = B1 + B2 such that B1 closes CSD gap and B2 closes RCV gap:
- CSD-component of CS = CSD − B1.
- RCV-component of CS = RCV − B2.
- Total CS = (CSD − B1) + (RCV − B2) = (CSD + RCV) − (B1 + B2).

Setting Total Extraction Damages (TED) = CSD + RCV:
- Total CS = TED − B = TED − (B1 + B2).

**Equational consistency PASSES.** Aggregate equation holds; decomposition is purely additive; flagship equation unchanged at aggregate level.

### §3.3 Boundary case — instruments spanning both

What if an instrument internalizes BOTH CSD and RCV components (e.g., a community-transition reserve that compensates past harm AND funds future-resource-substitution research)?

**Resolution:** the instrument's contribution to B is decomposed by its components. If a $X instrument is X% reparations-allocated and (1-X)% foreclosure-allocated, then it contributes X·$X to B1 and (1-X)·$X to B2. Aggregate B = B1 + B2 still holds.

### §3.4 Verdict on §3

PASSES. Decomposition is mathematically consistent with already-ratified flagship equation; boundary cases handled via component-decomposition.
