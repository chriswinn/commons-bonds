## §2. Pass-3 scope reminder

Pass 3 applies the audience pressure-test character set built for Ch 5's specific send context (Sandy Darity packet) + the broader publisher-facing pipeline. For each character the test asks:

- What does this character read for?
- What lands for this character in the post-Phase-C-β-follow-up prose?
- What flags for this character?
- Does the chapter hold this character's attention through the close?

Score per character on the canonical include-vs-exclude scale:

- **✓✓✓ INCLUDE** — strong yes; chapter lands decisively.
- **✓✓ INCLUDE** — yes; chapter lands.
- **✓ INCLUDE** — yes with minor reservations.
- **NEUTRAL ~** — neither alienated nor strongly engaged.
- **⚠ EXCLUDE** — may disengage; flag specific reason.
- **⚠⚠ EXCLUDE** — actively rejects; flag specific reason.
- **⚠⚠⚠ EXCLUDE** — disqualifying; cannot ship to Sandy with this character as Tier 1.

Tier-1 ⚠⚠⚠ at any character = STRUCTURAL REVISION NEEDED (re-fire Stage 2). Tier-1 EXCLUDE at lesser severity = SPOT-FIX NEEDED. Tier-2/3 EXCLUDE = recommended spot-fix unless author rules otherwise.
