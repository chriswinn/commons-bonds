# tools/rigor-passes/ — chapter-side + cross-essay rigor history

**What lives here:** rigor-pass artifacts for BOOK CHAPTERS and CROSS-ESSAY/corpus-wide work.
**What does NOT live here (since 2026-05-28):** per-essay rigor — that travels with the essay
package at `publishing/essays/<venue>/rigor/` (per-essay rigor consolidation pattern; see
`publishing/essays/_pipeline/per-essay-rigor-consolidation-pilot_2026-05-28.md`).

## Layout

- Top level — chapter-side + cross-essay artifacts still in active reference.
- `archive/` — historical eras (April-2026 framework-development sweeps, superseded protocol
  versions, pre-pipeline-v3 artifacts). Moved 2026-06-10 per

## Naming eras (both valid; do not rename old files)

- **Long-form (historical):** `commons_bonds_rigor_pass_<date>_<scope>_<pass>_v<semver>.md` —
  used through May 2026; the corpus of record.
- **Short-form (current, per-essay dirs only):** `pass-3-1-fact-check.md` etc., inside
  `publishing/essays/<venue>/rigor/` where the directory supplies the scope.
