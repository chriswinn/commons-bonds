## Input 1 — restoration $/acre → **ESTIMATE-LABEL (HIGH sub-floor materiality)**
- **TA:** ~$50,000–$200,000/acre → Ecosystem floor $12–50/ton (§11.6 L4649–4656).
- **External anchors, all far lower:** documented modern-mine reclamation ~**$12,000–15,500/acre** (Appalachian Voices, *Repairing the Damage*, ~$7.5–9.8B over ~633K acres); forest reforestation ~**$1,000–1,400/acre** (OSMRE / Powell River Project ARRI; Yale E360). Peer-reviewed (PLOS ONE; EPA): MTR stream/hydrology alteration is **not reliably reversible at any price**.
- **Verdict:** TA band is 3×–50× above every documented per-acre anchor. The gap is defensible *only as an explicit assumption* — that a genuine **functional ecological substitute** costs far more than the bond-funded regrade-and-revegetate minimum, with the non-substitutable remainder priced in Method 3 (which §11.6 L4653 already gestures at: "substitutable component only; non-substitutable component priced higher in Method 3").
- **Recommended treatment:** relabel as a framework estimate; state the functional-substitute-vs-reclamation assumption; cite the documented reclamation/reforestation anchors so the reader sees the gap. **AUTHOR EYE:** this is the soft underbelly a hostile reviewer attacks; the framing choice is yours.
- **Note:** pegged to documented reclamation cost, the Ecosystem floor would fall to ~$1–4/ton — immaterial to the combined anchor (Habitability dominates).

## Input 2 — McDowell disturbed acreage → **ESTIMATE-LABEL (MEDIUM sub-floor materiality)**
- **TA:** ~150,000 acres → Ecosystem floor.
- **No single published McDowell figure.** Regional anchor: ~**720,000 acres** Central-Appalachia *surface* mining 1985–2015 (Pericak et al. 2018, PLOS ONE; Duke/SkyTruth/Appalachian Voices), >1.5M with pre-1985. McDowell County = 535 sq mi ≈ **342,400 acres total**; 150K ≈ 44% of the county → plausible only as a cumulative century-long surface+underground footprint, implausible as surface-only.
- **Recommended treatment:** label as framework estimate with explicit apportionment basis. For a *sourced* number, pull WVDEP / OSMRE-AML permit acreage for McDowell FIPS 54047.

## Input 3 — $/displaced-resident → **ESTIMATE-LABEL, anchored (MEDIUM, direction-favorable)**
- **TA:** ~$200,000–$500,000/displaced-resident → Cohesion floor $17–42/ton (§11.6 L4666).
- **Real precedent:** Isle de Jean Charles, LA — first fully federally-funded whole-community relocation — **$48.3M for ~37 households (~$1.3M/household)** (LA Office of Community Development; Inside Climate News; Wikipedia). It prices physical resettlement, *not* social cohesion (residents report the rebuilt community doesn't function as the original — direct support for routing the non-substitutable cohesion good to Method 3).
- **Recommended treatment:** label as conservative engineering-substitute estimate; cite Isle de Jean Charles as anchor. The TA figure is **conservative** vs the ~$1.3M/household precedent.

## Input 4 — net out-migration ~50,000 → **CITE (+ light label) (LOW–MEDIUM, direction-favorable)**
- **TA:** ~50,000 → Cohesion floor.
- **Census basis:** 98,887 (1950 peak) → 19,111 (2020) = ~**80,000 decline** (US Census; WV Encyclopedia; QuickFacts). ~50K is a defensible *conservative* net-migration back-out (gross decline minus natural decrease); the full ~80K would raise the floor ~60%.
- **Recommended treatment:** cite Census endpoints; add a one-line decline-vs-net-migration distinction so ~50K doesn't read as a published migration figure.
