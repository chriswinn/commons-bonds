substance lands but the matrix-specified RELATION (the actual positioning move)
is not worked — e.g. Pigou is named but the "Pigou has no method; RCV has a
method" extend-relation is not shown. Those are flagged because a coverage check
reads them as present when the positioning value is not yet captured.
