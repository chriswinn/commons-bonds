## Constraints / disciplines

- **Mix target:** ~60% econ/policy big-idea / ~25% narrative-nonfiction-with-systems-thinking / ~15% wildcards (memoir-hybrid).
- **Avoid:** pure-literary agents (won't sell idea books fast); pure-academic agents (no trade advances); political-memoir agents (wrong frame).
- **Personalization-snippet discipline:** anchored to real public facts (recent deals, public interviews, acknowledgments-page mentions). No fabricated ties; no inflated claims.
- **Query letter discipline:** ≤400 words; one screen; no attachments unless explicitly invited; don't query simultaneous with exclusivity-only agents.
- **Wave 1 staggering:** spread sends across 1–2 weeks to learn from response patterns before committing all 8 Priority A queries.
