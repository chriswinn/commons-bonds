### §1.4 Pass 3.5 applicability decision

**The op-ed at 1,063w sits in the borderline zone for Pass 3.5 applicability.** Empirical anchors:

- Chapter scale (~5,000-10,000w): Pass 3.5 definitively applies. Ch 1 review surfaced 3 HIGH + 7 MEDIUM + 3 LOW. Ch 2/3/5/6/8/10 + foreign-affairs/atlantic-main/atlantic-ideas/noema/harpers/boston-review/100-barrel/nyrb-multi-book essays all received Pass 3.5.
- Essay scale (~3,000-6,000w): Pass 3.5 applies. $100 Barrel essay at 4,145w yielded 0 HIGH + 3 MEDIUM + 3 LOW.
- Aeon pitch scale (~300w): Pass 3.5 N/A per the prior Aeon V-D independent audit (§7.1: "not applicable to 301w pitch per Stage 3 template Audit-existing-prose mode").
- Op-ed scale (~1,000w): unprecedented. The pipeline doctrine codifies Pass 3.5 at "whole-chapter scale" with ~50K-80K tokens per chapter; the empirical anchors are all chapter or essay scale.

**Decision: Pass 3.5 APPLIES — at op-ed-scaled calibration.** Reasoning:

1. **Polarity rationale survives op-ed scale.** Pass 3.5 catches what per-paragraph Pass 3.2 cannot (whole-piece emotional-arc continuity; scene-anchor density across analytical pivots; cumulative-LLM-cadence residue; reader-engagement at analytical pivots). Op-eds have 9 paragraphs + analytical pivots + a reader-engagement burden at the news-peg-to-nut-graf-to-architecture-to-discipline-to-comparators-to-objection-to-existence-proof-to-call arc. The polarity discipline (restoration, not cutting) is relevant: V-D was actively compressed from ~5 em-dashes to 2, which is precisely the kind of cumulative chiseling that Pass 3.5 was designed to whole-piece-scan.

2. **V-D has not been through a formal Pass 3.5 anywhere in its lineage.** The locked Draft 2026-05-10 had informal Pass 3.3/3.4-equivalent baseline only (per its frontmatter); no Pass 3.5 fired. Neither alt nor V-D got a Pass 3.5. This audit is the first opportunity to fire Pass 3.5 at op-ed-pipeline scale; declining on length-grounds would propagate the precedent that op-eds skip Pass 3.5 indefinitely.

3. **Author flagged the prior audit under-scoped.** The author's explicit instruction was "run Pass 3.3 + 3.4 + 3.5 fresh." Decline-on-length-grounds would not honor the explicit instruction.

4. **Token economy is not a constraint.** Per `tools/memory/feedback_token_cost_not_a_constraint.md` (2026-05-28). The Pass 3.5 op-ed cost is ~10K-20K tokens — modest.

**Calibration to op-ed scale.** Empirical expected count for ~1,000w piece: 0 HIGH + 1-3 MEDIUM + 1-3 LOW = 2-6 findings total. (Chapter scale 13 findings on ~9,000w → linear-scaling on word-count gives ~1.5 findings on ~1,000w; allow for breadth via 2-6 calibration band.)

**Hard constraint (no-invented-factual-claims rule).** Pass 3.5 recommendations MUST be structure-only OR substrate-anchored against the canonical-facts inventory (`publishing/op-eds/_shared/canonical-facts/norway-mcdowell-inventory_2026-05-10.md`). Outside this perimeter, recommendations are structure-only.
