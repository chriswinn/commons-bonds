audience-pressure-test character set per
`tools/drafting-templates/audience-pressure-test-construction.md`. After

Sam Dresser; possibly an external reviewer Haselby loops in for second

This addendum: (i) reframes the audit's verdict weightings under the

## §1. The editor-facing-pitch reframe

### §1.1 What changes under the reframe

The character set is unchanged. What changes is **which character classes
are direct readers of the pitch vs. editor-projected from the pitch**:

| Tier | Characters | Pitch-read mode under reframe | Weight in pitch evaluation |
|---|---|---|---|
| Tier 1 | T1.1 Aeon editorial brain (Haselby/Dresser/Warburton); T1.2 Aeon reader (proxy — what editors think their readers want); T1.3 literary agent; T1.4 acquisitions editor | **DIRECT pitch-class readers** (editor-class; lit agent + acquisitions editor read at pipeline-downstream stage but still pitch-class) | **HIGH** — direct effect on submission outcome |
| Tier 2 | T2.5 Berggruen judge; T2.6 Aeon Philosophy Prize judges; T2.7 Mazzucato/Raworth cluster; T2.8 Coasean economist; T2.9 Ostrom-successor | **Editor-projected from pitch signals** — editor evaluates "does the author signal craft to handle these readers in the eventual essay?" | **MEDIUM** — indirect through editor projection |
| Tier 3 | T3.10 Black readers; T3.11 Indigenous readers; T3.12 McDowell County resident; T3.13 working-class reader; T3.14 first-gen immigrant | **Editor-projected from cultural-engagement signals** — editor evaluates "does the pitch demonstrate the author has thought about these audiences for the essay?" | **LOW** — engagement signals only at pitch scale |
| Adversarial | A1-A5 | **Editor-projected hostile-essay-reader projection** — editor evaluates "if this essay runs, will hostile readers tear it apart in ways that damage Aeon's reputation?" Pitch needs to SIGNAL author has analytical tool-kit; does not need to fully execute adversarial defense | **LOW-MEDIUM** — signaling axis only at pitch scale |

### §1.2 The implication for pitch craft

Under the editor-facing-pitch lens, the pitch needs to:

- **Make the thesis cleanly + directly** (T1.1 + T1.2 + T1.3 + T1.4 read
  directly; directness signals author conviction; hedge/mediation reads
  as lack-of-conviction at editor-evaluation scale).
- **Signal analytical-positioning craft** for Tier 2 + adversarial
  projection (V-E body's *"analytical lens"* + *"lands in arithmetic"* +
  *"not anti-extraction"* + asteroid-iron disarmers do this work).
- **Signal cultural-engagement commitment** for Tier 3 projection
  (V-E body's McDowell case + *"communities downstream"* + *"ecosystems
  that cannot send a bill"* do this work).

The pitch does NOT need to **fully execute** adversarial-defense or
cultural-resonance work — that's essay-level craft. Pitch-level
adversarial mediation can actively **damage** the editor-signal by
reading as defensive: an editor projects "if the author can't make this
claim cleanly in 300 words, how will they hold an essay together against
hostile readers?"

## §4. Methodology insight — editor-mediation reweighting for pitch-stage audits

### §4.1 The general principle

The audience-pressure-test character set in
`tools/drafting-templates/audience-pressure-test-construction.md` is
constructed for **end-reader-facing prose** evaluation (essay,
manuscript chapter, op-ed published in venue). Tier 2 + Tier 3 +
adversarial characters are direct readers of end-reader-facing prose;
the audit weights their verdicts as direct effects on prose reception.

Applying the same character set to **editor-facing pitch prose** without
reweighting introduces systematic lens-calibration drift:

- Adversarial findings get **over-weighted** (the hostile reader never
  reads the pitch; the editor projects future essay reception from
  pitch signals).
- Directness/conviction-signaling findings get **under-weighted** (the
  editor weights pitch directness as conviction-signal; defensive
  mediation in a pitch reads as author-lacking-conviction even when the
  same move would strengthen essay prose).

The primary Pass 3.3 + 3.4 + 3.5 audit on the Aeon Option E.2
contingency was not corrupted by this drift — its per-character data is
sound — but its aggregate net-effect synthesis under-credited
directness-loss costs and over-credited adversarial-mediation gains
relative to what an editor-facing-pitch lens would weigh.

### §4.2 Recommended editor-mediation reweighting for pitch-stage audits

When the artifact under audit is a **pitch** (or any other editor-facing
internal-pipeline artifact: query letter, cover letter, agent submission
letter, editor introduction email), apply the following reweighting:

- **Tier 1 acceptance characters (editor-class readers — actual + downstream
  pipeline editor-class readers):** Direct-reading weight = HIGH. These
  ARE the readers; verdicts apply at face value.
- **Tier 2 pipeline-strengthening characters:** Editor-projected from
  pitch signals. Weight = MEDIUM. Verdicts apply through editor's
  projection of "does the author signal craft to handle these readers in
  the eventual essay?" Within-verdict basis-deepening at Tier 2
  characters matters less than at Tier 1; pitch needs to signal
  analytical-positioning craft, not fully execute defense.
- **Tier 3 cultural-resonance characters:** Editor-projected from
  cultural-engagement signals. Weight = LOW. Pitch-scale engagement is
  signaling-only; full cultural-engagement work is essay-level.
- **Adversarial set:** Editor-projected hostile-essay-reader projection.
  Weight = LOW-MEDIUM. Pitch needs to signal analytical tool-kit; does
  not need to fully execute adversarial defense. Within-verdict basis-
  deepening at adversarial characters matters less at pitch scale than
  at essay scale.

When evaluating spot-fix candidates for editor-facing pitch prose,
explicitly check: "Does this fix improve essay-reader response while
damaging editor-reader directness signal?" If yes, the fix may be
correct at essay-scale and wrong at pitch-scale. Apply at essay-stage
(post-acceptance editing); do NOT apply at pitch-stage.

### §4.3 Carry-forward to discipline templates (RECOMMENDED; author ratifies separately)

The methodology insight above is general — it applies to all editor-
facing pitch audits, not just this one. Recommended carry-forward to:

- **`tools/drafting-templates/audience-pressure-test-construction.md`** —
  add a new section §"Editor-mediation reweighting for editor-facing
  prose" that codifies the reweighting principle above. Same character
  set; different weights when artifact is editor-facing pitch (or query
  letter, cover letter, agent submission letter, editor introduction
  email) vs end-reader-facing prose (essay, manuscript chapter,
  published op-ed).
- **`tools/memory/feedback_audience_aware_drafting_discipline.md`** —
  add a memory-entry amendment (Amendment D? candidate label) codifying
  editor-mediation reweighting as a pipeline-doctrine discipline.
  Specifically: at Stage 3 Pass 3.3 + 3.4 audits, classify the artifact
  as editor-facing-pipeline vs end-reader-facing-final-prose; apply the
  appropriate weighting rubric.

This addendum **recommends** the carry-forward but does NOT apply it
unilaterally — modifying the discipline-templates is scope-creep
relative to this audit's mandate. Author ratifies the recommendation in
a separate session if the carry-forward is desired.
