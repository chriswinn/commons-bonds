# Academic-Trade Press Editor Targets — Pipeline B

**Date drafted:** 2026-06-03  ·  **Status.** Exhaustive lane research sweep RATIFIED 2026-06-03 per Pipeline A + B exhaustive lane research chip; supersedes the prior tier-bound Pipeline B chip with count cap (~10-15). 73 verified editor rows across 16 academic-trade presses + 1 trade-publisher hybrid (Bloomsbury Academic).

**Purpose.** Per cascade-plan v2 §11, Pipeline B = academic-trade direct-to-editor queries, parallel-Plan-A territory (NOT Plan-B fallback). Half of the closest comps (Pistor / Patel & Moore / Sassen / Rodrik) deliberately chose academic-trade routes. Pipeline B optimizes against FPD v1.0.0 success criterion 4 (vocabulary adoption at scale in legal, regulatory, advocacy, and academic contexts) — the financial-advance + general-retail metrics that conventionally favor full trade are not the load-bearing criteria for a framework book.

**First Priority B target:** Duke University Press (register-perfect fit per Robinson / Hall / Sassen / Wendy Brown / Fred Moten lineage). Joe Jackson (Princeton) is the dominant single-editor target across the entire list (acquired Rodrik + Pistor).

## Sourcing approach

- Press editor pages (e.g., dukeupress.edu/About-Us, press.princeton.edu/our-team, hup.harvard.edu, mitpress.mit.edu, ucpress.edu, yalebooks.yale.edu/about-us/editors).
- Press catalogs (recent acquisitions visible in season catalogs).
- Acknowledgments pages of comp / adjacent titles.
- Press blogs ("Five Questions with …" interview series at UCP; "Meet Our Editors" at Yale; Editor Spotlight at UC Press).
- LinkedIn current-employment verification.
- Industry coverage (Publishers Weekly, Publishing Trends, Chronicle of Higher Education, Inside Higher Ed, The Bookseller, Shelf Awareness, Manuscript Works newsletter).

## Tier heuristics

Same priority A/B/C structure as Pipeline A:
- **Priority A** — direct comp lineage; portfolio overlaps Mazzucato/Pistor/Sandel/Tooze/Christophers/Susskind/Patel/Moore/Sassen/Rodrik register; verified recent acquisition.
- **Priority B** — strong adjacency.
- **Priority C** — broader fit; lower direct match.

## Submission protocol per press

- **Duke UP:** Single-editor query via Book Proposal Submission Form. Do NOT cross-submit. If misfit, editor will internally route.
- **Princeton UP:** Email proposal directly to subject editor. Full proposal per [PUP Proposal Guidelines](https://assets.press.princeton.edu/authors/PUP-PROPOSAL-GUIDELINES.pdf).
- **Harvard UP:** Email to relevant acquisitions editor OR submissions_hup@harvard.edu. Full proposal required. Does NOT accept first-time queries via referral-only.
- **MIT Press:** Initial **inquiry email** first. Do NOT send full manuscript/sample chapters/images unless invited.
- **UC Press:** Single query email to ONE editor + CV + sample chapter(s) + proposal. Response timeline 8-10 weeks. **Cross-querying prohibited.**
- **Yale UP:** Editor-direct query email + 1-2 page proposal + 1-2 sample chapters. Format: `<firstname>.<lastname>@yale.edu`. UK uses `<firstname>.<lastname>@yaleup.co.uk`.
- **Chicago UP:** Editor-direct email confirmed for Zimmerman + Doskow. Format: `<firstinitial><lastname>@uchicago.edu`. Proposal + sample per [press page](https://press.uchicago.edu/resource/publishing_with_chicago.html).
- **Cambridge UP:** Submission via the "Find an Editor" portal at [cambridge.org/authorhub](https://www.cambridge.org/authorhub/). Cambridge book proposals require synopsis + chapter outline + sample chapters + market/competition analysis + author CV.
- **Oxford UP:** Submission via [global "Find an editor" contact](https://global.oup.com/academic/contactus/editorial/). OUP requires proposal + chapter outline + sample chapters; trade division prefers an agent submission.
- **Stanford UP:** Query via SUP author portal.
- **Cornell UP:** cornellup_acq@cornell.edu or editor-direct.
- **JHU Press:** Editor-direct.
- **Toronto UP:** Editor-direct.
- **Brookings Press:** Editor-direct; verify Finan currency.
- **Russell Sage:** Director-of-publications gatekeeper.
- **Hart / Bloomsbury Academic:** Editor-direct.
- **Polity Press:** Editor-direct (Knight or Owers).
- **Routledge:** Editor-direct via [editorial contacts](https://www.routledge.com/contacts/editorial); subject-by-subject routing.
- **Penn / Columbia / Michigan / NYU / Minnesota:** Editor-direct via press editor pages.
