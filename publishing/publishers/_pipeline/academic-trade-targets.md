# Pipeline B — Academic-Trade Acquisitions Editor Targets

**Date drafted:** 2026-06-03  ·  **Date modified:** 2026-06-03

**Status.** Initial population (13 named editors across 9 presses). Per cascade-plan v2 §11 Pipeline B specification (RATIFIED 2026-06-01).

**Purpose.** Ranked target list of acquisitions editors at academic-trade university presses who acquire the kind of non-partisan econ-policy framework nonfiction that *Commons Bonds* belongs in — Pistor / Patel & Moore / Sassen / Rodrik / Mazzucato / Susskind / Milanović register, with serious public-intellectual reach into legal, regulatory, advocacy, and academic readerships.

**Pipeline B optimizes FPD v1.0.0 success criterion 4** (vocabulary adoption at scale in legal, regulatory, advocacy, and academic contexts). Half of the closest comps (Pistor / Patel & Moore / Sassen / Rodrik) deliberately chose academic-trade routes. Pipeline B is parallel-Plan-A territory, not consolation; for a framework book whose anchor success criterion is vocabulary diffusion, Pipeline B may deliver higher EV than Pipeline A. See [cascade-plan v2 §11 Pipeline B](../../essays/_pipeline/cascade-plan_v2_2026-05-24.md) and [§1 point 9](../../essays/_pipeline/cascade-plan_v2_2026-05-24.md) for the strategic frame.

**Pipeline B fires on its own trigger (~Aug 2026)** — not gated by Pipeline A. Direct-to-editor queries; optional agent representation post-offer if author wants to convert. Realistic query → publication timeline 12–22 months optimistic, 15–20 months median (per cascade-plan v2 §6.5).

## Sourcing approach

- Press editorial-team / staff pages (most academic-trade presses publish acquisitions-editor names + portfolios publicly).
- Acknowledgments pages of recent acquisitions in the framework comp-title space.
- Editor LinkedIn profiles + press-release announcements for career-history confirmation.
- Press catalogs + series listings to verify the editor's recent acquisitions match the framework register.
- **Avoid:** generic-submissions email addresses (Pipeline B is about NAMED editors, per cascade-plan §11).

## Sourcing discipline (no-invented-facts rule per cascade-plan §1 point 10)

- Every editor row cites a verifiable public source for the role + recent acquisition (press staff page, acknowledgments page, LinkedIn, press release).
- Where current employment is verified via press staff page within the last 12 months, marked "Verified 2026-06."
- Where current employment is verified only via earlier press release or LinkedIn, marked "Verification pending fresh staff-page confirmation."
- Recent-acquisition citations cite a specific book title + publisher + year that the editor verifiably acquired or edited. Acknowledgments-page confirmation is the gold standard; press-staff-page attribution + LinkedIn cross-check is the working minimum.

## Mix target (within Pipeline B's 10–15 editor envelope)

- ~50% economics + political-economy direct (Joe Jackson, Tovbis, Ditchik, Zimmerman, Woods, Dreesen, Swallow)
- ~30% political/social theory + race-economy lineage (Wissoker, Berger, Schneider, Brandt, Frucht)
- ~20% wildcards: humanities-leaning trade-list bridges (Flannery-McCoy)

## Discipline

- **Verify each editor's current employment** before query firing (academic-trade editors typically move every 3–5 years; staff-page check within 90 days of firing).
- **Confirm acknowledgments-page citations** — the gold standard for editor-acquisition attribution is the acknowledgments page of the cited book, not press marketing copy. Where this verification step has not yet been performed (rows 1, 6, 7, 8, 10, 11, 12), the row Notes flag "via press staff page + LinkedIn; acknowledgments-page verification pending."
- **Track per-editor query state in a sibling `query-tracker.md`** (modeled on `publishing/agents/_pipeline/query-tracker.md`) once Pipeline B query-firing begins (~Aug 2026).
- **Note submission-protocol variance** — academic-trade presses range from "proposal-first via form" (Cambridge, OUP) to "subject-area-editor direct query OK" (Princeton, Yale, Chicago). Per-press protocol noted in each row's Notes column.
- **Imprint-orientation veto** (carried forward from Pipeline A discipline): Pipeline B editors at partisan-coded series/imprints (rare in academic-trade but possible) get author sign-off against FPD v1.0.0 non-partisan stance before offer-acceptance.

## Ranking rule

- **Priority A (~5):** strongest direct comp-overlap; editor's acquired list contains ≥1 of the eight framework comp titles OR editor sits at the cross-pipeline alignment point (e.g., Zimmerman / Phenomenal World Books).
- **Priority B (~5):** strong fit; editor's portfolio register matches framework's three load-bearing dimensions (institutional injustice + race-economy + non-partisan policy posture); recent acquisitions in the framework neighborhood without being direct comp-overlap.
- **Priority C (~3):** good fit at the margin; newer-tenure editors building a list (Woods); slower-cycle presses (Swallow); institutional-state caveats requiring verification (Brandt).

Fire queries in waves matching priority — A first (1–5), B second (6–10), C third (11–13).
